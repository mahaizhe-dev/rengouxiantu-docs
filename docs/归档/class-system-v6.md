## 一、职业总览

| 职业 | 核心属性 | 战斗特色 | 定位 |
|------|---------|---------|------|
| 武僧 | 根骨 (Constitution) | 更多重击 | 肉盾/持续输出 |

| 阵法师 | 福缘 (Fortune) | 增强宠物 + BUFF | 辅助/宠物流 |

| 职业 | 天赋效果 |
|------|---------|
| 武僧 | 重击概率 +10%（基础10% → 20%） |
| 剑修 | 法术连击概率 +8%（基础0% → 8%） |
| 阵法师 | 宠物同步率 +10%（T4: 65% → 75%） |

### 每级成长

每个职业每升1级，对应核心仙缘属性 +1 点：
- 武僧: 根骨 +1/级
- 剑修: 悟性 +1/级
- 阵法师: 福缘 +1/级

## 三、技能设计

### 武僧技能

| 技能 | 类型 | 效果 | CD |
|------|------|------|-----|
| 金刚掌 | 主动 AoE | 150% ATK，近战范围，最多4目标，20%减速 | 5s |
| 伏魔刀 | 主动 吸血 | 100% ATK，吸血30% | 8s |
| 金钟罩 | 被动 护盾 | HP<50%时自动触发，DEF×400% 护盾，持续8s | 60s |

**100点根骨被动 —— 金刚之力**:
- 效果: 重击时，重击值（HeavyHit）按双倍计算
- 公式: `重击伤害 = ATK + HeavyHit × 2`（而非默认的 `ATK + HeavyHit`）
- 视觉: 重击时角色金光大闪，拳头残影
- 注意: 是重击值×2，不是整个伤害×2

### 剑修技能

| 技能 | 类型 | 效果 | CD |
|------|------|------|-----|
| 御剑诀 | 主动 AoE | 150% ATK，剑气前方扇形，最多4目标 | 5s |
| 破妄斩 | 主动 AoE | 200% ATK，小范围扇形60°，最多4目标 | 9s |
| 剑阵降临 | 被动复合 | 见下方详细说明 | - |

**被动 —— 剑阵降临（双效果）**:
1. **剑阵效果**: 每次释放技能（御剑诀/破妄斩），自动在目标区域降下1道剑阵
   - 剑阵伤害: 60% ATK，小范围 AoE
   - 独立暴击判定
2. **御剑回神**: 击杀回血效果 ×2（放大装备的 killHeal 属性）

**100点悟性被动 —— 剑意**:
- 效果: 法术连击概率 +10%（总计 8%天赋 + 悟性加成 + 10% = 约18%）
- 额外效果: 法术连击触发时，额外降下1道剑阵（即连击时降2道剑阵）
- 视觉: 角色周围漂浮剑影环绕

**法术连击机制说明**:
- 技能释放后，概率立即重新施放一次（同一技能）
- 连击的那次有独立暴击判定
- 不消耗额外 CD
- 不可连锁（连击不能再触发连击）

### 阵法师技能

| 技能 | 类型 | 效果 | CD |
|------|------|------|-----|
| 天雷阵 | 主动 DoT | DEF × 80%/秒 持续伤害，范围内敌人，持续6s | 12s |
| 聚灵阵 | 主动 BUFF | 自身+宠物 ATK +30%，持续10s | 18s |
| 护法阵 | 主动 护盾 | DEF × 400% 护盾 | 25s |

**阵法师技能特色**:
- 所有技能（包括输出技能天雷阵）都与防御系数相关
- BUFF 技能同时作用于自身和宠物（未来扩展：也可加队友）
- 装备偏好: DEF = 攻防一体的核心属性

**100点福缘被动 —— 灵犀**:
- 效果: 聚灵阵激活期间（10s），宠物进入狂暴状态
- 狂暴效果: 宠物攻击速度 +50%
- 视觉: 宠物变红发光（小红兽），攻击特效加强
- 设计呼应: 与现有"犬魂狂暴"（宠物死后玩家暴走）形成镜像机制

## 四、DPS 计算（T7 基准）

### 计算条件

| 参数 | 值 |
|------|-----|
| 玩家ATK | 600 |
| 敌方DEF | 400 |
| 暴击率 | 60% |
| 暴击伤害 | 400% |
| 重击值(HeavyHit) | 200 |
| 宠物等级 | T4 |
| CalcDamage公式 | ATK² / (ATK + DEF) |

### 基础计算

```
CalcDamage(600, 400) = 600² / (600+400) = 360
期望暴击倍率 = 0.6×4.0 + 0.4×1.0 = 2.8
基础DPS（普攻1.0s）= 360 × 2.8 = 1008/s
```

### 武僧 DPS

```
普攻DPS: 1008/s
重击DPS:
  100点被动前: 重击伤害 = CalcDamage(600+200, 400) = 512, 暴击期望 = 512×2.8 = 1434
    概率20%, 贡献 = 1434 × 0.2 = 287/s
  100点被动后: 重击伤害 = CalcDamage(600+400, 400) = 571, 暴击期望 = 571×2.8 = 1600
    概率20%, 贡献 = 1600 × 0.2 = 320/s（提升仅 +33/s）

技能DPS:
  金刚掌: 360×1.5×2.8 = 1512, CD5s → 302/s
  伏魔刀: 360×1.0×2.8 = 1008, CD8s → 126/s（+吸血 302×0.3=91/s回复）
  金钟罩: 被动，不计入DPS

总DPS（单目标）= 1008 + 287 + 302 + 126 = 1723/s（无100点被动）
总DPS（单目标）= 1008 + 320 + 302 + 126 = 1756/s（有100点被动）

根骨20%加成后:
  DEF+20% → 减伤提升（防御向收益）
  HP+20% → 生存提升
  综合DPS ≈ 2082/s（含根骨属性对重击概率的加成）
```

### 剑修 DPS

```
普攻DPS: 1008/s
剑阵DPS:
  每次技能触发1道: 360×0.6×2.8 = 605
  御剑诀: 605/5s = 121/s
  破妄斩: 605/9s = 67/s
  剑阵总贡献: 188/s

技能DPS:
  御剑诀: 360×1.5×2.8 = 1512, CD5s → 302/s
  破妄斩: 360×2.0×2.8 = 2016, CD9s → 224/s

法术连击（18%概率，含100点被动）:
  额外技能DPS: (302+224) × 0.18 = 95/s
  额外剑阵(100pt): 188 × 0.18 = 34/s

总DPS = 1008 + 188 + 302 + 224 + 95 + 34 = 1851/s
悟性加成后（+技能伤害%）: ≈ 2526/s
```

### 阵法师 DPS

```
普攻DPS: 1008/s
天雷阵:
  DEF×80%/s × 6s = 玩家DEF(假设500含根骨) × 0.8 × 6 = 2400总伤
  CD12s → 200/s（不走CalcDamage，直接DEF系数伤害）
  实际 DPS ≈ 161/s（考虑敌方减伤）

宠物DPS:
  同步率: 65% + 10%天赋 = 75%
  宠物ATK = 600 × 0.75 = 450
  CalcDamage(450, 400) = 238, 暴击期望 = 238 × 2.8 = 667/s
  聚灵阵期间: 宠物ATK = 780×0.75 = 585 → CalcDamage(585,400) ≈ 347 × 2.8 = 972/s
  100点狂暴(攻速+50%): 972 × 1.5 = 1458/s（聚灵阵10s/CD18s窗口期）
  宠物平均DPS ≈ 837/s

总DPS = 1008 + 161 + 837 = 2006/s
```

## 六、设计原则备忘

1. **100点被动**: 单一效果，无"必定"，基于累积强化
2. **法术连击 vs 双倍伤害**: 连击（重新施放）体验优于双倍数字
3. **阵法师**: DEF = 攻防一体属性，BUFF 机器定位
4. **剑修**: 高爆发但脆皮，击杀回血是唯一续航
5. **宠物狂暴**: 绑定技能 CD（聚灵阵），可预测、可操作
6. **重击值×2**: 只翻倍 HeavyHit 数值，不翻倍整个伤害
