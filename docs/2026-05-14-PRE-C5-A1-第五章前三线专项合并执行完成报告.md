| 写请求不自动补发 | `BatchSet:Save` 失败只报错，不重试 | ✅ 已有 |

### 5.2 状态机行为详解

```
connected ──(fail×1)──► unstable ──(fail×3)──► disconnected
    ▲                                              │
    └──────────(success×RESTORE_THRESHOLD=2)────────┘
```

- 第1次失败 → unstable + `connection_unstable` 事件（轻提示）
- 第2次失败 → 仍 unstable，无新事件（宽限期）
- 第3次失败 → disconnected + `connection_lost` 事件（阻断）
- 恢复需连续2次成功 → connected + `connection_restored` 事件
