# 第五章 · 怪物设定

> 状态：基础规划稿  
> 用途：整理第五章各区域的基础怪物和基础掉落。  
> 原则：第五章各区域按章节结构配置怪物，多数主区为 `1种普通怪 + 1种精英怪 + 1个BOSS`，`藏经书阁` 只放精英和 BOSS，`镇魔深渊` 与 `四封剑台` 只放 BOSS。  
> 关联文档：`docs/设计文档/第五章-太虚之殇.md`

## 一、整体规则

| 项目 | 内容 |
|------|------|
| 怪物主线 | `宗门废墟异变 -> 镇魔深渊供血 -> 四封剑台终局` |
| 区域规则 | 多数主区固定 `1普通怪 + 1精英怪 + 1BOSS`，`藏经书阁` 为精英+BOSS，`镇魔深渊` 为双BOSS结构 |
| 入口守卫 | `裂山门遗址` 左右两侧各刷新 `1` 只 `95级` 王级 Boss `护山石傀`，作为双路线开启前的入口守卫 |
| BOSS定位 | 六大主区 BOSS 大多为太虚宗高层，与第四章“太虚宗年轻一代俊才坐镇八阵”形成代际对照；`洗剑寒池` 例外为护池灵兽，`镇魔深渊` 的 Boss 为外来大魔 |
| 怪物气质 | 左路偏战斗/锻造/生活残骸，右路偏寒池/碑林/书阁传承崩坏 |
| 深渊规则 | `镇魔深渊` 不放普通怪和精英怪，改为区域 Boss + 魔帅 Boss 的双 Boss 结构 |
| 终局结构 | `太虚剑宫` 不作为常规刷怪区，内部四个封剑台只放对应神剑 Boss |
| 特殊区规则 | `万剑回廊` 暂时不刷怪，只保留后续特殊 Boss 和强化线入口定位 |

## 五、基础掉落设计

### 5.1 基础规则

| 项目 | 内容 |
|------|------|
| 掉落范围 | 本文只记录 `灵韵 / 食物 / 金币 / 经验 / 基础随机装备` |
| 其他掉落 | 章节专属材料、残剑、戒指、令牌、特殊道具不在本文件展开 |
| 金币结算 | 按 `掉落设计.md` 的等级、阶级、境界公式自动结算 |
| 经验结算 | 按 `掉落设计.md` 的等级、阶级、境界公式自动结算 |
| 等级差衰减 | 玩家等级高于怪物 `10级以上` 时，经验、金币、食物不掉落 |
| 食物拟合原则 | 按第四章实际战斗掉落覆盖宠物经验约 `39%` 的比例拟合第五章；若后续调整 `100~120` 宠物经验，需按当前刷图模型重新核算平均目标 |
| 食物掉落口径 | 普通怪只掉 `仙骨`，精英怪只掉 `妖兽精华`，BOSS 只掉 `龙髓` |
| 食物基础档 | 普通怪固定 `8% 仙骨×1`，精英怪按等级从 `12%` 递增到 `30%` 掉落 `妖兽精华×1`，主区 BOSS 按章节进度递增龙髓掉率 |
| 精英食物阶梯 | `95区精英 12%`，`100区精英 16%`，`105区精英 20%`，`110区精英 24%`，`115区精英 27%`，`119区精英 30%` |
| BOSS食物阶梯 | `95级BOSS 20%`，`100级BOSS 40%`，`105级BOSS 60%`，`110级BOSS 80%`，`115/119级BOSS 必掉1个` |
| 终局BOSS食物 | `裂渊屠血将 50% 龙髓×1`，`镇渊魔帅 必掉龙髓×2`，`四仙剑 必掉龙髓×3` |
| 章节装备阶段 | 第五章目标装备阶段为 `T9 / T10`，但因全局窗口规则，`91~100` 仍会产出 `T7~T9`，`101~120` 才进入 `T8~T10` |
| 普通怪装备 | `10%` 掉落随机装备，使用完整 Tier 窗口，品质上限固定 `purple` |
| 精英怪装备 | `100%` 掉落随机装备，使用完整 Tier 窗口；参考第四章配置，前段精英上限为 `orange`，`Lv.96+` 后可提升到 `cyan` |
| 区域BOSS装备 | `100%` 掉落随机装备，只取当前窗口最高 `2` 个 Tier；参考第四章，从 `green~orange` 逐步抬升到 `blue~cyan` |
| 终局BOSS装备 | `镇魔深渊` 与 `四封剑台` 作为终局 farming 节点，装备进一步收束到 `固定T10` |
| 品质门槛 | `T9`：`Lv.91` 解锁橙，`Lv.96` 解锁青；`T10`：`Lv.106` 解锁橙，`Lv.111` 解锁青 |
