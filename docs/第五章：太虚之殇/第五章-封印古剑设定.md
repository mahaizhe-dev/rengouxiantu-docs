# 第五章 · 封印古剑设定

> 用途：整理第五章四把 `封印古剑` 的独立掉落、武器属性与特效。  
> 原则：本稿只处理 `封印态`，即四把剑掉落时的 `T10青·灵器武器` 版本；解封后的 `圣器态` 另开文档。

---

## 一、统一规则

| 项目 | 设定 |
|------|------|
| 装备名 | `封印古剑·诛仙 / 陷仙 / 戮仙 / 绝仙` |
| 部位 | `武器` |
| 品质 | `青色灵器` |
| 阶位 | `T10` |
| 掉落来源 | 四仙剑各自独立掉落自己的封印古剑 |
| 掉率 | `0.5%（独立掉落）` |
| 主属性规则 | 四把剑统一为 `T10青武器主属性` |
| 主属性数值 | `攻击 +189` |
| 副属性规则 | 每把剑固定 `3条副属性`，全部为确定值，不走随机波动 |
| 特效规则 | 每把剑固定 `1条武器特效`，封印态特效只做 `单条核心被动` |
| 后续成长 | 使用第五章解封材料后，升为 `圣器态`，属性与特效另行设计 |

### 设计口径

- 参考第四章龙极武器口径：`统一主属性 + 3副属性 + 1条核心特效`。
- 四把封印古剑虽然同属 `T10青·灵器武器`，但属性方向必须明显分开。
- `诛仙` 偏暴击爆发。
- `陷仙` 偏生命与稳态输出。
- `戮仙` 偏重击压制。
- `绝仙` 偏持续进攻与收割。
- 封印态特效不能做成终局完全体，必须给解封后的 `圣器态` 留空间。

## 二、四把封印古剑

### 2.1 封印古剑·诛仙

| 项目 | 内容 |
|------|------|
| 掉落 Boss | `诛仙剑` |
| 品质 / 阶位 | `T10青·灵器` |
| 主属性 | `攻击 +189` |

| 定位 | 偏稳态持续战，兼顾生存与稳定暴击。 |

### 2.3 封印古剑·戮仙

| 项目 | 内容 |

| 品质 / 阶位 | `T10青·灵器` |

| 副属性 | `根骨 +18`、`重击 +104`、`攻击 +19.5` |
| 特效 | `裂地`：重击时，额外追加一次 `100%攻击力` 的伤害。 |
| 定位 | 偏重击压制与稳定破防，最像第四章裂地线。 |

### 2.4 封印古剑·绝仙

| 副属性 | `击杀回血 +39`、`攻击 +19.5`、`攻击 +19.5` |

## 四、执行规格

### 4.1 全局口径

- 四把封印古剑都按 `T10青·灵器武器` 执行。
- 主属性统一：`攻击 +189`
- 每把剑固定：`3条副属性 + 1条特效`
- 本文只定义 `封印态`
- 特效全部按 `常驻被动` 处理，不做主动释放
- `附加伤害` 默认视为额外伤害事件，不再触发本武器自己的二次附加

### 4.2 封印古剑·诛仙执行规格

| 项目 | 设定 |

| 副属性3 | `攻击 +19.5` |
| 特效名 | `诛心` |
| 触发条件 | `造成暴击时` |
| 特效效果 | `追加一次 30%攻击力 的伤害` |
| 触发频率 | `满足条件即触发` |
| 冷却 | `无` |
| 额外规则 | `该附加伤害不再二次触发诛心` |

## 五、执行口径

### 5.1 已经定死的内容

- 四把封印古剑全部是 `T10青·灵器武器`
- 四把封印古剑主属性统一 `攻击 +189`
- 四把封印古剑都是 `0.5%` 独立掉落
- 每把剑固定 `3条副属性 + 1条特效`
- 四把剑的特效方向必须分别对应：
  - `诛仙 = 暴击联动`
  - `陷仙 = 生命联动`
  - `戮仙 = 裂地重击`
  - `绝仙 = 攻击联动`

### 5.2 这一份文档暂不处理

- 解封后的 `圣器态` 属性
- 解封后的 `圣器态` 新特效
- 四把剑的图鉴加成
- 四把剑的出售价格与分解产物
