# 第五章 · 特殊装备设定

> 状态：规划稿  
> 用途：整理第五章怪物掉落的特殊装备、终局独立掉落、属性、特效与图鉴加成。  
> 原则：延续第四章“章节专属 + 终局独立稀有”的结构；第五章重点是 `4件青色套装件 + 8件散件 + 帝尊伍戒 + 四把封印古剑`。  
> 关联文档：`docs/设计文档/第五章-怪物设定.md`、`docs/设计文档/第五章-太虚之殇.md`

---

## 一、整体结构

| 项目 | 内容 |
|------|------|
| 常规专属数量 | `12` 件章节专属装备 |
| 终局独立掉落 | `帝尊伍戒`、`四把封印古剑` |
| 覆盖部位 | `头盔 / 鞋子 / 项链 / 腰带 / 戒指 / 肩膀 / 衣服 / 披风` |
| 套装结构 | 第五章只做 `2` 对 `2件套` |
| 套装要求 | `头盔 + 腰带` 为生命套；`鞋子 + 肩膀` 为防御套 |
| 套装品质 | `4` 件套装件全部为 `T10青` |
| 掉落逻辑 | 严格按 Boss 等级推进：`95入口T9橙 -> 95/100主区T9青 -> 105/110主区T10橙 -> 115/119深层Boss -> 120深渊 -> 120终局独立稀有` |

### 设计口径

- `护山石傀（镇派石兽）` 作为入口守卫 Boss，补一件 `T9橙` 特殊装备。
- `问剑长老·裴千岳`、`洗剑霜鸾` 两位主区 Boss 各补 `1件 T9青` 特殊装备。
- `铸剑长老·韩百炼`、`守碑长老·石观澜` 两位中层 Boss 各补 `1件 T10橙` 特殊装备。
- `别院院主·宁栖梧` 掉落 `T10青` 鞋子套件与 `T10橙` 戒指散件，两件走 `共享1%` 掉率池。
- `藏经阁主·温素章` 掉落 `T10青` 头盔套件与 `T10青` 衣服散件，两件走 `共享1%` 掉率池。
- `裂渊屠血将` 掉落 `T10青` 腰带套件与 `T10青` 肩膀套件，两件走 `共享0.5%` 掉率池。
- 披风是特殊装备，直接挂在 `镇渊魔帅·噬渊血犼` 身上。
- `帝尊伍戒` 改为 `噬渊血犼 + 四仙剑` 各自 `0.5%` 独立掉落。
- 四仙剑各自独立掉落对应的 `封印古剑`，掉率统一 `0.5%`。

### 掉率说明

- `护山石傀（镇派石兽）` 作为入口 Boss，先给 `T9橙`，作为第五章特殊装起点。
- `问剑 / 寒池` 两位主区 Boss 直接推进到 `T9青`。
- `地炉 / 碑林` 两位中层 Boss 接入 `T10橙`，且 `T10橙` 统一按 `1%` 处理。
- `宁栖梧`、`温素章` 的两件专属分别走 `共享1%` 掉率池。
- `裂渊屠血将` 的两件专属走 `共享0.5%` 掉率池。
- 披风作为特殊装备，直接挂在 `镇渊魔帅·噬渊血犼` 身上，不分配给前中段 Boss。
- `帝尊伍戒` 改为 `0.5%` 独立掉落，不占用其他专属池。
- 四仙剑各自的 `封印古剑` 也统一改为 `0.5%` 独立掉落。
