## 一、执行范围

第五章新增账号级神器 `诛仙阵图`，沿用现有神器系统框架，包含：

1. `9块神器碎片`
2. `3×3九宫格激活`
3. `账号级属性加成`
4. `神器专属挑战Boss`
5. `独立神器被动技能`
6. `统一被动技能形态`
7. `仙图录神器页接入`
8. `黑市碎片上架`
9. `存档 / 读档 / 新角色同步 / 自测流程`

原则：

- 不新开一套神器系统，复用 `ArtifactSystem / ArtifactUI / AtlasSystem`
- 第五章神器仍为 `账号级神器`
- 技能为 `被动触发技能`
- 剑阵技能为 `120秒内置冷却`
- 所有职业共用同一技能形态，不做太虚专属强化

### 6.1.1 挑战场地与生成口径

- 沿用前三章神器 Boss 的 `EnterBossArena` 逻辑
- 点击挑战后，玩家被传送至 `独立副本竞技场`

- 场内只生成 `1名神器Boss：太虚宗主·司空玄胤`
- 不在大世界原地图直接刷新，不与第五章主线区域怪物混刷
- 挑战胜负后，沿用现有 `ExitBossArena` 口径返回原场景并恢复备份状态

### 6.3 战斗定位

- 定位为第五章神器专属 Boss
- 强于四把仙剑单体挑战
- 强调 `宗主 / 血池 / 四剑阵图 / 连续施法`
- 技能池由 `四仙剑部分技能 + 宗主专属归宗技` 组成
- Boss 本体直接施放四剑系技能，不额外召唤四把实体剑

### 6.4 技能总规则

| 项目 | 设定 |
|------|------|
| Boss形态 | 中心固定站位 Boss，可原地转向，可短距换面，不追击绕场 |
| 技能来源 | 掌握 `诛仙 / 陷仙 / 戮仙 / 绝仙` 的部分技能 |
| 节奏特点 | 至少存在 `1组重复连续释放技能`，二、三阶段连续施压明显增强 |
| 场地机制 | 中央血池长期存在，随阶段提升活跃度 |
| 生成限制 | 不刷常规小怪，不额外生成四把实体剑，不拆成多Boss协同 |

### 6.5 技能清单

#### 常驻被动

| 技能名 | 类型 | 描述 | 用处 |
|------|------|------|------|
| 剑宗统式 | 常驻被动 | 司空玄胤每完成一次完整技能后，下一次技能有 `35%` 概率进入 `连式` 状态：在本次技能结算后 `0.45秒` 内，原技能再追加一次缩短版重复施放。缩短版只保留核心伤害区，不再完整铺场。 | 形成“连续释放”的宗主特征。 |
| 血池养剑 | 常驻被动 | 中央血池常驻存在。Boss每次施放一次 `四剑系技能`，获得 `1层养剑`；满 `4层` 后，下一次四剑系技能自动强化为 `归宗式`，并清空层数。 | 把四仙剑技和宗主专属强化施法串起来。 |

#### 一阶段常规技能

| 技能名 | 来源 | 描述 | 用处 |
|------|------|------|------|
| 一字诛线 | 诛仙剑 | 向正前方打出一条超长直线剑气，延迟短，伤害高。 | 基础斩线压制。 |
| 寒镜回锋 | 陷仙剑 | 向一个方向放出剑光，命中镜区后折返一次，折返段延迟明显。 | 制造读图与二次换位。 |
| 熔环外推 | 戮仙剑 | 以Boss为中心放出一圈外扩火环，经过区域留下短时灼地。 | 逼迫离开中心。 |
| 禁疗血雾 | 绝仙剑 | 指定区域生成血雾，站在其中治疗效果显著下降。 | 压制回血与站桩。 |
| 连式·诛线 | 宗主专属重复技 | 司空玄胤连续释放 `3次` 窄化版一字诛线，每次间隔 `0.35秒`，角度轻微偏转。 | 一阶段的重复连续释放技能。 |

#### 二阶段新增技能

| 技能名 | 来源 | 描述 | 用处 |
|------|------|------|------|
| 吸血锁链 | 绝仙剑 | 向目标连出血链，短延迟后开始吸血；离开锁链范围可挣断。 | 逼迫主动拉位。 |
| 十字绝路 | 诛仙剑 | 先纵斩后横斩，两次交叉斩线在Boss脚下展开。 | 强化中心危险度。 |

#### 三阶段新增技能

| 技能名 | 来源 | 描述 | 用处 |
|------|------|------|------|
| 八荒焚路 | 戮仙剑 | 向多方向放出熔线，场地留下大面积灼烧带。 | 大幅压缩站位。 |
| 绝命蚀界 | 绝仙剑 | 全场进入短时血界，仅保留少量安全点。 | 三阶段血线压迫。 |
| 万剑归宗 | 宗主终结技 | 司空玄胤短蓄力后，依次快速释放强化版 `十字绝路 -> 寒镜回锋 -> 八荒焚路 -> 绝命蚀界`，最后在中心补一记大范围竖斩。若处于 `连式` 状态，则只追加后一半缩短版。 | 终局爆发技能。 |

### 6.6 阶段结构

#### 第一阶段

- 生命区间：`100% ~ 70%`
- 使用技能：
  - `一字诛线`
  - `寒镜回锋`
  - `熔环外推`
  - `禁疗血雾`
  - `连式·诛线`
- 阶段特征：
  - 以 `四仙剑单技能拆解版` 为主
  - 通过 `连式·诛线` 先建立“重复连续释放”认知
  - 血池只做轻度共鸣，不直接爆发

#### 第二阶段

- 生命区间：`70% ~ 40%`
- 转阶段技能：`血池灌体`
  - 宗主短暂悬空，血池翻涌扩张
  - 宗主获得一次 `满层养剑`
  - 结束后，宗主进入二阶段
- 使用技能：
  - 保留一阶段全部技能
  - 新增 `吸血锁链`
  - 新增 `十字绝路`
  - 新增 `四式归宗`
- 阶段特征：
  - 血池进入活化状态
  - `血池养剑` 的积累更快
  - `四式归宗` 成为中段主要爆发源

#### 第三阶段

- 生命区间：`40% ~ 0%`
- 转阶段技能：`入魔归宗`
  - 宗主强制击退玩家
  - 血池范围扩大并持续翻涌 `3秒`
  - 宗主直接进入 `归宗态`
- 使用技能：
  - 保留 `一字诛线 / 十字绝路 / 吸血锁链 / 四式归宗`
  - 新增 `八荒焚路`
  - 新增 `绝命蚀界`
  - 新增 `万剑归宗`
- 阶段特征：
  - 场地压力最大
  - Boss连式概率提升
  - 连续释放明显增多

### 6.7 技能循环逻辑

- 一阶段循环：
  - `一字诛线 -> 寒镜回锋 -> 熔环外推 -> 禁疗血雾`
  - 每 `2轮` 插入一次 `连式·诛线`
- 二阶段循环：
  - `十字绝路 -> 吸血锁链 -> 四式归宗 -> 一字诛线 / 禁疗血雾`
  - 若 `血池养剑` 满4层，则立即优先插入一次 `四式归宗`
- 三阶段循环：
  - `四式归宗 -> 八荒焚路 -> 绝命蚀界 -> 吸血锁链`
  - 大技能结束后有概率进入 `连式`
  - `万剑归宗` 作为定时大招插入循环

### 6.8 Boss技能实现备注

1. 宗主掌握的四仙剑技能，均采用 `缩小版 / 宗主版`，不完全复制原Boss整套机制。
2. `连式` 状态是实现连续施法的核心，不需要为每个技能额外写一套完整重复版本。
3. 宗主专属归宗技优先复用四仙剑现有危险区与伤害模板，不额外召唤四把实体剑。
4. Boss 挑战按前三章神器口径，始终只生成 `1名Boss`，不拆分副体。

## 七、神器被动技能

### 7.1 普通形态：四剑诛灭

| 项目 | 数值 |
|------|------|
| 类型 | `被动触发神器技能` |
| 冷却 | `60秒内置冷却` |
| 触发条件 | `施放主动技能时，若内置冷却完成且范围内有敌人，则自动触发` |
| 目标锁定 | `锁定范围内最近敌人位置；无目标时不触发（不空放）` |
| 作用范围 | `半径2格` |
| 命中段数 | `4段` |
| 段间隔 | `0.18秒` |
| 第一段伤害 | `攻击力 × 100%` |
| 第二段伤害 | `攻击力 × 150%` |
| 第三段伤害 | `攻击力 × 200%` |
| 第四段伤害 | `攻击力 × 250%` |
| 总伤害 | `攻击力 × 700%` |

#### 7.1.1 触发流程

1. 角色施放一次 `主动技能`
2. 若 `四剑诛灭` 内置冷却(60秒)完成，且6格范围内有敌人，则锁定最近敌人位置
3. 四把仙剑PNG从不同高角度蓄力停顿后急速斜插砸入目标区域：
   - `诛仙`
   - `陷仙`
   - `戮仙`
   - `绝仙`
5. 每段独立结算伤害

#### 7.1.2 命中规则

- 判定方式：`同一圆形区域重复命中`
- 只有 `施放主动技能` 才可触发本神器被动
- `普攻`、持续掉血、反伤、受击回击不触发本神器被动
- 每段都可独立暴击 / 重击
- 第四段高伤段必须允许吃暴击
- 每段都可触发正常受击反馈
- 本技能自身不递归触发同类神器技能
- 本技能附加伤害不额外再触发一次四剑下砸
- 本技能不占技能栏，不增加额外主动按钮

技能描述：

- 四把仙剑依次从四角微斜砸向同一片小范围区域
- 每次命中造成一段伤害
- 四段只做纯伤害，不附加易伤、不附加额外 debuff

### 7.2 技能表现重点

- 本技能重心在 `四段依次下砸的视觉表现`
- 实际判定仅需做成 `半径2格的小范围AOE`
- 四段都落在同一片区域内，允许轻微错位，制造围杀感
- 四段伤害递增，第 `4段` 作为视觉与数值上的收束重击

## 十、账号级、存档与流程方案

### 10.1 账号级生效原则

- `诛仙阵图` 的激活进度为 `账号级`
- `属性加成` 为 `账号级`
- `Boss击败状态` 为 `账号级`
- `技能解锁状态` 为 `账号级`
- 任意角色推进后，账号内其他角色同步享有进度

### 10.2 数据归属

- 数据存入 `account_atlas`
- 不写入单角色独立存档

### 10.3 流程链

1. 击杀指定Boss掉落 `阵图残符`
2. 碎片进入背包后触发即时保存
3. 玩家打开神器UI，消耗金币、灵韵、碎片激活格子
4. 激活格子后立即写回 `account_atlas`
5. 九格全开后，出现 `唤醒神器` / `挑战宗主` 按钮
6. 击败神器Boss后，写入 `bossDefeated / passiveUnlocked`
7. 角色侧获得神器被动技能
8. `仙图录` 同步显示最终状态

### 10.4 新角色同步

- 新角色创建后，需要走一次 `AtlasSystem.PostLoadSync()`
- 确保神器加成、道印加成、神器状态全部同步到账
