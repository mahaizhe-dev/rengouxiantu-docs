# T阶段属性构成拆分表

## 1. 口径

这份表按“**单属性上限视角**”统计。

- `T阶段` 取该阶段尾端等级。
- `贡献比例` 按该属性的当前总值占比计算。
- 主表只统计 **代码已接线、且有明确上限的确定性养成**。
- 主表**不混入**随机副属性、品质波动、锻造/洗练、灵性、图录、称号、酒、宠物书、活动/竞速道印这类会让分母随 build 大幅变化的项；这些统一放到后面的“条件/深化轴”。
- `ATK/HP/福缘/悟性/根骨/体魄` 中，`天道问心` 本身带分支互斥，所以表格是“该属性单独拉满时的贡献”，**不是一套角色能同时拿到全部分支上限**。
- 当前主线内容实际到 `T9 / Lv100`；`T10-T11` 目前主要是数值延长，没有新的常规主线养成系统。

主表纳入的系统：

- 角色成长：初始面板、升级成长、境界突破奖励
- 装备主轴：按 `T1-T11` 白板 12 槽主属性骨架
- 天道问心
- 悟道树
- 阵营凝X丹
- 封魔体魄丹
- 海神柱
- 文王八卦盘
- 境界大师道印
- 福缘果

主表未纳入但后文会补的系统：

- 图录
- 称号
- 专属位法宝替换
- 装备副属性/洗练/灵性/附灵/套装
- 龙酒
- 宠物赐主书
- 条件道印
- 徽章

## 4. 职业修正

主表故意按 **职业中性口径** 写，避免把三职业搅在一起。

如果要落到具体职业，再加下面这层：

- `太虚`：每级 `+1悟性`；如果计入同职业道印，再额外 `+1悟性 / 10级`
- `罗汉`：每级 `+1根骨`；如果计入同职业道印，再额外 `+1根骨 / 10级`
- `镇岳`：每级 `+1体魄`；如果计入同职业道印，再额外 `+1体魄 / 10级`

以 `T9 / Lv100` 为例：

- 太虚额外 `+100悟性`，若算道印则 `+110`，等于再多 `+20%~22%` 技能伤害
- 罗汉额外 `+100根骨`，若算道印则 `+110`，等于再多 `+20%~22%` 防御乘区
- 镇岳额外 `+100体魄`，若算道印则 `+110`，等于再多 `+20%~22%` 生命乘区和 `+30~33/s` 回复

### 5.2 专属位法宝替换

`5.1` 里的“默认专属悟性”只是白板骨架。

实际从 `T3` 开始，专属位会逐步被阵营法宝或龙极令替换，所以这一槽位不能一直按“固定悟性”理解。

#### 阵营法宝

四个阵营法宝都走 `仙缘公式`：

- `主属性 = floor(tier × qualityMult) × 5`
- `血海图` 替换为 `悟性`
- `浩气印` 替换为 `福缘`
- `青云塔` 替换为 `根骨`
- `封魔盘` 替换为 `体魄`

首通确定性品质下，专属位主属性如下：

| 阶段 | 声望/Tier | 首通品质 | 替换后的专属主属性 |
| - | -: | -: | -: |
| T3 | 声望1 / T3 | 蓝 | 15 |
| T4 | 声望2 / T4 | 蓝 | 20 |
| T5 | 声望3 / T5 | 紫 | 30 |
| T6 | 声望4 / T6 | 紫 | 35 |
| T7 | 声望5 / T7 | 紫 | 45 |
| T8 | 声望6 / T8 | 紫 | 50 |
| T9 | 声望7 / T9 | 橙 | 65 |

这意味着：

- `悟性路线`：`T3` 起就不是白板 `15悟性` 以下，而是很快被更高品质法宝顶替
- `根骨/体魄/福缘路线`：`T3` 起就能把专属位直接改造成对应四维槽
- `T7-T9` 时，这一槽位本身已经是四维主轴，不再只是“补一点悟性”

#### 龙极令

`T9` 还存在攻击型专属位替换：`龙极令`。

- 来源：第四章打造
- 品质：固定 `cyan`
- 公式：`2 × T9倍率17 × 品质1.8 = 61.2`，按当前标准公式取整后为 `61 ATK`

所以到 `T9`，专属位存在两类完全不同的 build 方向：

- `四维法宝路线`：把专属位做成 `悟性/福缘/根骨/体魄`
- `龙极令路线`：把专属位改成 `攻击`

主表之所以没有把这部分强行混进统一分母，是因为它会随着 build 目标在 `攻击` 和 `四维乘区` 之间切换。

## 7. 总结

如果只看确定性主轴：

- `T1-T3`：攻防血仍然是 `角色成长 + 装备骨架` 主导。
- `T4`：挑战丹药第一次重塑面板，尤其 `ATK/DEF/HP/回复`。
- `T5`：上宝逊金钯 + 体魄丹把 `根骨/体魄` 从边轴抬成主轴。
- `T7`：海神柱 + 八卦盘接入后，`海神柱` 主打平面，`八卦盘` 主打四维。
- `T9+`：没有新的常规主线系统，比例变化开始主要由等级成长和装备 Tier 本身推动。

如果把图录、套装、副属性、龙酒、宠物赐主也算进来，后期真正的 build 会比主表更偏向“**四维乘区 + 条件常驻轴**”，而不是纯白板主属性。
