### 怪物进阶结构

- **小怪**：每寨独有种族（枯木精/岩蟾/苍狼/赤甲蝎/蛇骨妖/赤焰妖/蜃妖）
- **BOSS**：全部为妖王级，2~3阶段战斗，越往内战斗机制越复杂
- **精英巡逻**：三片荒漠各有沙暴妖兵（妖尉→妖将→妖帅），主线需击杀100个
- **枯木守卫**：黄天寨门卫（固定型BOSS，地刺远程攻击）
- **猪二哥**：隐藏神器BOSS，Lv.70帝首领，3阶段

### 主线任务（sand_zone）

1. **扫平外围** — 击败枯木妖王+岩蟾妖王 → T5紫装三选一
2. **深入腹地** — 击败苍狼妖王 + 沙漠精英×100 → T2技能书三选一
3. **肃清妖患** — 击败赤甲妖王+蛇骨妖王 → T6紫装三选一
4. **斩妖除魔** — 击败烈焰狮王 + 沙漠精英×100 → T2技能书三选一
5. **兵临城下** — 击败蜃妖王 + 沙漠精英×100 → T7紫装三选一
6. **解除封印** — 解锁第一寨大门

### 剧情脉络

沙漠中九寨妖王联盟盘踞，以黄天大圣·沙万里为首。玩家从最外围的废寨出发，逐寨攻破，每寨妖王有独特战斗风格。三片荒漠间有精英妖兵巡逻。最终在第一寨对决帝级妖王沙万里。隐藏BOSS猪二哥（神器级）潜伏于神器竞技场。

### 跨章挑战系统

- **血煞盟·沈墨**（T1~T8）：血印 debuff 机制，叠层持续掉血
- **浩气宗·陆青云**（T1~T8）：结界领域机制，持续伤害区域
- 挑战不属于特定章节，随等级解锁

### 设计特点

- **八卦阵法**：8个阵法区域按等级梯度递增（Lv.66→99），每阵对应一种元素属性，由太虚宗上古弟子坐镇
- **封印推进**：坎阵免费开放，后续七阵（艮→乾）逐步解锁，每击杀当前阵BOSS 10次解锁下一阵传送
- **四方龙种岛**：四角最高等级区域（Lv.100·大乘初期），4条帝级龙族BOSS，各有3阶段战斗和独特龙鳞材料
- **天罡四柱**：乾阵门卫（固定型BOSS，地刺远程），分别铭刻「心正」「意诚」「身端」「道明」
- **境界跨度**：元婴后期(Lv.66) → 大乘初期(Lv.100)，横跨化神、合体、大乘三个大境界
- **新增种族**：水傀、岩俑、雷魂、残影、焦骨、土俑、蛊毒、石柱等8种新种族修正

1. **破坎阵** — 击败沈渊衣×10 → T7橙装三选一（解锁艮阵）
2. **破艮阵** — 击败岩不动×10 → T8紫装三选一（解锁震阵）
3. **破震阵** — 击败雷惊蛰×10 → 九转金丹×1 + 龙髓×1（解锁巽阵）

### 设计特点

- **仙界失落感**：第七章不是进入完整仙界，而是到达已经破碎的仙界边缘。所有建筑、道路、传送阵都应表现“曾经辉煌，现在只剩残骸”。
- **从回家到远渡**：第六章情绪是回到起点并发现故土被改写；第七章情绪转为离开故土，进入真正的仙域废墟。
- **中心安全区很小**：分界地只承担传送、基础补给和主线 NPC，不应像中洲或两界村那样繁华，避免削弱荒芜感。
- **四域非线性但有门槛**：四域可从分界地通往，但推荐按炎→雪→岩→雷的主线顺序推进；后续可用四枚界域印或四圣封印控制深处开放。
- **四圣兽作为终盘锚点**：四圣兽不只是普通 Boss，而是旧仙界四方秩序残存的最后封镇。击败或唤醒四圣兽后，可开启更深层仙界/第八章入口。
- **人仙段承接**：第七章承接第六章终盘 `人仙1阶`，主线目标推进至 `人仙9阶`，为第八章 `地仙1阶` 留出口。

### 预排原则

- **一章一大仙阶**：第八章从地仙开始，每章完整覆盖一个大仙阶的 9 个小阶和 20 级等级段。
- **一章一仙装主轴**：第八至十三章分别对应仙3~仙8；第十四章以仙9为常规主轴，仙10只作为终局预留，不提前泛滥投放。
- **主题分层**：万妖国与青丘之上负责“妖族线”；蓬莱蜃影负责“海外仙踪与蜃影幻境”；归墟与昆仑负责“仙界沉没与登天源头”；古天庭与离恨天负责“旧天庭真相与终局执念”。
- **妖族不是插曲**：万妖国偏妖族军国、诸部朝贡与力量压迫，青丘之上偏狐族幻术、情劫与旧约，两章共同构成仙界崩坏中“妖族旧盟”的完整视角。
- **蓬莱承上启下**：蓬莱蜃影位于妖族线与归墟/昆仑线之间，用海上仙山和蜃影幻境解释“长生仙界旧路为何被篡改”。
- **材料不复用前章货币**：每个大仙阶建议有自己的章节材料与转数仙劫丹，避免第六章 `谪仙令`、第七章 `界海令` 等旧货币长期滚雪球。
- **终局章控制仙10**：仙10不建议作为完整普通掉落阶位，应作为离恨天终局的特殊打造、终局 Boss 低概率、高成本升级或账号级长线目标。

| ID | 名称 | 防御 | 坐标 | 说明 |

**傀儡共性**：
- `category = "boss"`（体型 1.3，视觉突出）
- `hp = 999999`，`atk = 0`，`speed = 0`
- `stationary = true`，`passive = true`，`isTrainingDummy = true`
- 不掉落、不给经验/金币、不计入击杀统计
- 永不死亡，受击后立即回满血

#### DPS 面板

玩家进入训练场坐标范围后，自动弹出 **DPS 追踪面板**，实时显示输出数据：

| 功能 | 说明 |
|------|------|
| 实时 DPS | 基于最近 5 秒滑动窗口计算 |
| 最高 DPS | 本次训练的峰值 DPS |
| 累计伤害 | 自上次重置以来的总伤害 |
| 当前目标 | 锁定最近攻击的傀儡名称 |
| 空闲冻结 | 3 秒无输出自动冻结 DPS 数值（保留读数） |
| 自动隐藏 | 离开训练场坐标范围后面板自动关闭 |
| 手动重置 | 切换攻击目标时自动重置统计 |

#### 三层保护机制

训练傀儡通过三层独立保护确保**永不死亡、不产生任何游戏副作用**：

| 层级 | 位置 | 机制 |
|------|------|------|
| 第一层 | `Monster:TakeDamage()` | 拦截伤害，仅触发受击闪白 + `monster_hurt` 事件（供 DPS 面板统计），然后立即回满血 |
| 第二层 | `GameEvents.monster_death` | 前置守卫 `if isTrainingDummy then return`，阻止掉落/经验/击杀记录 |
| 第三层 | `CombatSystem` 击杀效果 | 同上守卫，阻止击杀特效和浮动文字 |

#### 技术备注

| 项目 | 说明 |
|------|------|
| 区域检测 | DPS 面板使用**坐标直接判定**（`TRAINING_BOUNDS = {x1=3, y1=72, x2=16, y2=79}`），不依赖 `GetZoneAt()`，避免与 `mz_base` 区域的 `pairs()` 迭代顺序不确定性冲突 |
| `combatActivated_` 标志 | 区分"因进入区域自动显示"和"因攻击傀儡而显示"，防止区域检测抖动导致面板闪烁 |
| BOSS 类别兼容 | 傀儡使用 `category = "boss"` 会被 `BOSS_CATEGORIES` 匹配，但因三层保护，Spawner 冷却/击杀计数/存档统计均不受影响 |
| 配置文件 | `MonsterTypes_training.lua`（傀儡定义）、`tianji.lua`（区域/刷怪点/装饰） |
