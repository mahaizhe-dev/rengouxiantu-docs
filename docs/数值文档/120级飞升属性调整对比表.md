# 120级飞升属性调整对比表

## 1. 口径说明

本文只比较裸属性，不含装备、丹药、图鉴、称号、神器、洗髓、职业被动、体魄/根骨等额外系统。

旧属性口径来自当前代码：

1. 初始属性：`生命100 / 攻击15 / 防御5 / 回复1`
2. 等级成长：每级 `生命+15 / 攻击+3 / 防御+2 / 回复+0.2`
3. 境界突破：当前 `GameConfig.REALMS` 配置

新属性设计口径：

1. 初始属性不动。
2. 等级成长调整为：每级 `生命+15 / 攻击+3 / 防御+2 / 回复+0.3`。
3. 优先调整境界突破奖励。
4. 生命/攻击保持接近 `5:1`。
5. 防御/攻击从前期约 `0.6~0.67`，逐步过渡到大乘巅峰与谪仙初期的 `0.8`。
6. 回复/生命逐步收敛到 `2%/秒`。
7. `120级大乘巅峰 -> 谪仙初期` 是最大单次突破。
8. 谪仙初期突破奖励固定为：`生命+500 / 攻击+100 / 防御+80 / 回复+10`。

目标基线：

| 阶段 | 生命 | 攻击 | 防御 | 回复 |
| --- | ---: | ---: | ---: | ---: |
| 新 Lv120 大乘巅峰 | 4500 | 900 | 720 | 90 |
| 谪仙初期突破 | +500 | +100 | +80 | +10 |
| 谪仙初期基线 | 5000 | 1000 | 800 | 100 |

## 5. 设计结论

1. 新等级成长为 `15 / 3 / 2 / 0.3`。
2. Lv120 大乘巅峰基线调整为 `4500 / 900 / 720 / 90`。
3. 谪仙初期突破固定为 `+500 / +100 / +80 / +10`。
4. 谪仙初期最终基线为 `5000 / 1000 / 800 / 100`。
5. 最终比例为 `生命/攻击=5.00`、`防御/攻击=0.80`、`回复/生命=2.00%`。
6. 前期生命、攻击、防御阶段终点基本不动；主要变更集中在回复成长、化神后防御、合体后防御、大乘期防御与回复。
7. 谪仙初期仍是最大单次突破，满足飞升质变要求。
