# 第一章掉落设定表
更新时间：2026-07-03

## 说明

- 本文按当前工作树的第一章配置核对整理。
- 普通怪物掉落、主线奖励、触发战斗奖励、每日小钻风、虎王试炼炉分开统计，避免把一次性/合成/日常奖励误算进常规 Boss 掉落。
- 第一章世界掉落池 `ch1` 当前实际只有 `金条`，所以第一章所有 `world_drop = ch1` 都等价于掉 `金条`。

### 主线奖励备注

| 项目 | 说明 |
| --- | --- |
| 第一章主线链 | 任务链 ID 是 `town_zone`，章节名是“两界村主线”，当前 `schemaVersion = 2` |
| 装备奖励槽位 | 只会从 `helmet`、`shoulder`、`belt`、`boots`、`ring1`、`ring2`、`necklace` 中抽取 |
| 不会出现的部位 | 主线 `3选1` 不会出 `weapon`、`armor`、`cape`、`treasure`、`exclusive` |
| 选择方式 | `count = 3` 的含义是生成 3 件候选装备，玩家最终只领取其中 1 件 |

### 线上存档影响

| 玩家状态 | 上线后影响 |
| --- | --- |
| 新角色 / 未创建第一章主线状态 | 从新第 1 步 `击败拦路猪妖` 开始，可获得 `练气丹 x1`、`金条 x1` |
| 旧存档第一章未完成 | 加载时 `town_zone.currentStep` 自动 `+1`，保持原本正在做的目标不变，避免因插入新第一步而倒退或错位 |
| 旧存档正在猪三哥步骤 | 迁移后仍然指向猪三哥；完成后发放新的 `T2 purple` 装备 3 选 1，不再发旧材料包 |
| 旧存档已完成第一章主线 | 保持完成，`currentStep` 迁移到新链末尾，并继续同步虎啸林封印为已解除；不会补发新增第 1 步奖励 |
| GM 重置第一章主线 | 重置后从新第 1 步开始，完整体验新版 5 步流程 |
