## 1. 基础规则

1. `Lv120` 为飞升前常规等级上限。
2. 大乘巅峰无法常规升入谪仙，需要通过飞升渡劫玩法。
3. 渡劫成功后，玩家进入 `谪仙1阶`。
4. 后续成长使用 `仙阶`，共 `81阶`。
5. 仙阶结构为 `9个大仙阶 × 每个大仙阶9阶`。
6. 从 `谪仙` 开始，每个大仙阶解锁 `20级` 等级上限。
7. 等级成长不再按 `Lv121-Lv300` 单独配置固定翻倍值，改由当前 `仙体` 的每级成长模板决定。
8. 飞升会获得第一个仙体；仙体会从 `Lv1` 开始重算等级成长属性，不是只影响 `Lv120` 以后。
9. 当前等级成长模板锚点：`凡人 HP+15 / ATK+3 / DEF+2 / 回复+0.3`，`仙人之体 HP+25 / ATK+5 / DEF+3 / 回复+0.5`。
10. 每一阶都需要消耗 `仙力` 或指定珍稀材料突破，属性增量按第 3 节执行。
11. 飞升后的角色等级经验曲线按第 4 节执行；经验来源、升级入口、等级是否必须绑定仙阶突破，后续单独确定。

### 3.1 玩家进阶属性规则

基准取 `谪仙1阶` 明确突破值：

```text
HP+500 / ATK+100 / DEF+80 / 回复+10
```

大仙阶系数采用固定表，不做指数递增：

| 大仙阶 | 系数 |
|--------|----:|
| 谪仙 | 1.0 |
| 人仙 | 1.2 |
| 地仙 | 1.4 |
| 天仙 | 1.6 |
| 玄仙 | 1.8 |
| 金仙 | 2.0 |
| 太乙金仙 | 2.5 |
| 大罗金仙 | 3.0 |
| 混元大罗金仙 | 4.0 |

突破属性公式：

```text
大仙阶1阶属性 = 谪仙1阶基准属性 * 大仙阶系数
小阶属性 = 当前大仙阶1阶属性 * 0.2
每个大仙阶总属性 = 当前大仙阶1阶属性 * (1 + 8 * 0.2) = 当前大仙阶1阶属性 * 2.6
```

完整仙阶线总境界倍率：

```text
大仙阶系数和 = 1.0 + 1.2 + 1.4 + 1.6 + 1.8 + 2.0 + 2.5 + 3.0 + 4.0 = 18.5
总境界倍率 = 18.5 * 2.6 = 48.1
```

### 3.2 玩家300级属性核算

本节只核算角色本体四维，不含装备、丹药、图鉴、称号、神器、洗髓、职业被动、根骨体魄等额外系统。

等级成长口径：

```text
等级成长累计 = (当前等级 - 1) * 当前仙体每级成长
凡人每级成长 = HP+15 / ATK+3 / DEF+2 / 回复+0.3
仙人之体每级成长 = HP+25 / ATK+5 / DEF+3 / 回复+0.5
```

### 4.1 经验设计口径

1. 后120级经验以普通 BOSS 经验作为统一锚点。
2. 普通 BOSS 经验按当前怪物经验规则计算：`基础经验 * boss经验倍率5 * 仙阶奖励系数`。
3. 仙阶奖励系数继续沿用第 5.3 节规则：`奖励系数 = 1.0 + 境界order * 0.2`，仙阶 `order = 22 + 仙阶总序`。

6. 段内奖励系数按 20 级均匀映射到 9 个仙阶：`段内仙阶 = ceil(段内等级序号 * 9 / 20)`。
7. 逐级经验生成时，单级升级经验按 `普通BOSS经验(等级, 仙阶总序) * 目标普通BOSS数` 反推；若出现小数，最终配置取整数。

本节只作为数值观察口径，暂不代表最终投放。

| 谪仙1阶 | 6.0 | +1.0 | 飞升后首阶质变 |

观察口径：

1. 大乘中期到大乘巅峰，每个小境界约 `+0.3` 属性系数。
2. 大乘巅峰到谪仙1阶，直接 `+1.0`，用于体现飞升质变。
3. 谪仙1阶从 `6.0` 起步后，再进入后续仙阶递增规则。

### 5.2 属性系数规则

1. `谪仙1阶` 怪物属性系数定为 `6.0`。
2. 同一大仙阶内，每提升 `1小阶`，怪物属性系数 `+0.5`。
3. 从一个大仙阶九阶进入下一个大仙阶一阶时，怪物属性系数 `+1.0`。
4. 怪物属性系数只作用于 `HP / ATK / DEF`。
5. 怪物速度不吃仙阶属性系数，仍按怪物阶级倍率与种族修正处理。

公式：

```text
属性系数 = 6.0 + (大仙阶序 - 1) * 5.0 + (阶位 - 1) * 0.5
```

### 5.3 奖励系数规则

奖励系数暂时沿用当前代码口径：

```text
奖励系数 = 1.0 + 境界order * 0.2
```

仙阶体系中，暂按 `大乘巅峰 order=22` 之后继续顺延：

```text
仙阶奖励order = 22 + 仙阶总序
```

因此 `谪仙1阶` 的奖励系数为 `1.0 + 23 * 0.2 = 5.6`，与当前代码中 `谪仙初期` 的奖励系数保持一致。
