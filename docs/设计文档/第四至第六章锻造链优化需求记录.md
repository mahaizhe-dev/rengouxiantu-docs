## 五、装备升级与背包打造

底层两种模式已经存在，本轮不新增锻造执行器，也不改变原有语义。

| 模式 | 数据字段 | 实际规则 |
|---|---|---|
| 装备升级 | `equip_weapon`＋`replace_weapon` | 必须装备指定武器，成功后原位替换 |
| 背包打造 | `bag_consume`＋`add_to_bag` | 不要求穿戴，从背包扣除材料，产物进入背包 |

本轮 UI 只加强可见区别：

- 装备升级显示 `装备升级`，固定提示“需先装备指定武器，打造后原位替换”。
- 背包打造显示 `背包打造`，固定提示“无需穿戴，材料从背包扣除，产物进入背包”。
- 五把龙极武器页面新增帝尊肆戒的背包数量显示与不足提示。
- 龙极令页面新增帝尊肆戒的背包数量显示与不足提示。
- 真龙盔作为龙神熔炉新增背包打造页签。
- 龙魂令和龙王令成功提示按实际产物显示，不再使用旧“龙极令升级”描述。
- 武器升级继续显示当前装备与产物对比；背包打造显示拥有数量/需求数量。

`ForgeSystem` 继续负责最终校验和原子扣除，UI 本地判断仅用于按钮状态和提示。

## 八、验收标准

1. 五把龙极武器缺少帝尊肆戒时不可打造，材料齐全后仍原位替换当前武器。
2. 真龙盔只读取背包，不要求装备极龙盔，产物属性和图鉴加成正确。
3. 龙极令缺少帝尊肆戒时不可打造。
4. 龙魂令不再消耗龙极令，固定消耗仙人精血5、太虚剑令1000、帝尊伍戒1。
5. 四把解封古剑、道藏圣衣、深渊圣氅的仙人精血均为5。
6. 龙王令消耗影子水晶6、谪仙令1000、仙尊壹戒1，不消耗前代龙令。
7. 龙王令与龙魂令使用完全相同的龙息技能配置，只提升装备属性。
8. 武器升级与背包打造在 UI 上有明确标签和说明。
9. 第六章五件法宝只有 T11 红色 / 圣器可以收录，其他品质均不可收录。
10. 全锻造流水线配方数量为29，所有配方均通过 `Check -> Execute`。
11. 生成失败、材料不足或背包无空间时不发生扣费。
12. Maker Lua LSP 工作区诊断为 `0 errors`。
13. `git diff --check` 通过。
