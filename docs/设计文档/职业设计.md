## 一、职业总览

## 三、技能设计

### 罗汉技能（已同步代码 — 2026-03-21）

| 技能 | 类型 | 解锁 | 效果 | CD |
|------|------|------|------|-----|
| 金刚掌 | 主动 AoE | Lv.3 凡人 | 150% ATK，前方圆形1.5格，最多5目标，20%减速3秒 | 5s |
| 伏魔刀 | 主动 吸血 | Lv.6 凡人 | 100% ATK，前方矩形2×1格，最多3目标，吸血100%伤害值 | 8s |
| 金钟罩 | 被动 护盾 | 练气初期 | HP<50%时自动触发，DEF×4.0 护盾，持续8s | 60s |
| 龙象功 | 被动 蓄力 | 筑基初期 | 普攻+1层，重击+2层，满18层释放龙象气：1倍重击真实伤害，AoE 4格，最多8目标（可暴击） | — |

> **龙象功详细机制**（代码: `SkillSystem.DragonElephantAddStacks` / `TriggerDragonElephantQuake`）:
> - 叠层来源: 普攻+1层，重击+2层
> - 满层触发: 18层 → 自动清零 → 释放龙象气
> - 伤害公式: `floor(ATK + HeavyHit) × (1 + heavyDmgBonus)`，与重击公式相同，**无视防御**
> - 暴击: 走玩家暴击判定
> - AoE: 以玩家为中心4格范围，最多命中8个目标（全技能最高）
> - 技能伤害%: **不吃**悟性技能伤害加成（因为走重击公式而非 CalcDamage）

### 太虚技能（全剑主题重设计 — 2026-04-14 v3）

| 技能 | 类型 | 解锁 | 效果 | CD |
|------|------|------|------|-----|
| 破剑式 | 主动 AoE | Lv.3 凡人 | 150% ATK，三柄剑窄扇形飞出，最多3目标，**削弱攻击力10%持续3s** | 5s |
| 奔雷式 | 主动 AoE+DOT | Lv.6 凡人 | **150%** ATK砸击 + 插剑持续电击（**10% ATK/s × 5s，真实伤害**），面前圆形1.5格，最多5目标 | 5s |
| 御剑诀 | 被动 减伤 | 练气初期 | 3柄灵剑环绕，每柄抵挡1次攻击30%伤害；剑消耗后每20s恢复1柄，释放技能额外恢复1柄 | — |
| 一剑开天 | 被动 N次触发 | 筑基初期 | 每释放**8次**技能（含连击），从天召唤巨剑：**300%** ATK，面前圆形1.5格，最多5目标，**双倍击杀回血** | — |

> **数值被动（每级成长）**: 太虚每升1级 +1 悟性（代码: `CLASS_DATA.taixu.passive.wisdomPerLevel = 1`），对应 `Player:_RecalcStatsCache()` 中 `classWisdom = level × wisdomPerLevel`，计入 `GetTotalWisdom()`。与罗汉"每级+1根骨"对称。

> **v2 变更记录 (2026-04-05)**:
> - 剑魂不灭(免死复活) → **御剑诀**(持续减伤)：风格从"死了再说"改为"稳定防护"
> - 剑阵降临: 瞬发50%单次伤害 → **持续电击10%/s×5s，真实伤害无视防御**，每tick独立暴击
> - 一剑开天: 维持 **300%** ATK（高倍率爆发定位）

> **与罗汉对标设计说明**:
>
> | 对比维度 | 罗汉 | 太虚 |
> |---------|------|------|
> | 技能1 (Lv.3) | 金刚掌: 150%ATK, 5目标, 减速 | 破剑式: 150%ATK, 3目标, 窄扇形 |
> | 技能2 (Lv.6) | 伏魔刀: 100%ATK, 3目标, **吸血** | 奔雷式: **150%**ATK砸击+DOT, 5目标 |
> | 被动1 (练气) | 金钟罩: DEF×4护盾 | 御剑诀: 3剑环绕，各挡30%伤害 |
> | 被动2 (筑基) | 龙象功: 18层→AoE真伤 | 一剑开天: 8次技能→300%AoE巨剑 |
> | 核心属性收益 | 根骨→重击概率/伤害+防御 | 悟性→技能伤害%+连击概率 |
> | 生存手段 | 吸血+护盾（持续续航） | 双倍击杀回血+御剑诀（减伤防护） |
> | 定位差异 | 持续真伤+高生存 | 技能爆发+DOT叠加+被动巨剑 |
>
> **关键差异**:
> - 伏魔刀(100%+吸血) vs 奔雷式(150%+DOT) — 罗汉稳定续航，太虚砸击+持续电击
> - 金钟罩(DEF×4护盾) vs 御剑诀(3×30%减伤) — 罗汉一次性大护盾，太虚多次小减伤
> - 一剑开天(被动8次→300%AoE)是太虚的核心爆发引擎，奔雷式的高频DOT是持续DPS基底

**破剑式详细机制**:
- 三柄剑同时飞出，60°窄扇形，圆形预筛范围2.5格
- 每柄剑独立命中，最多命中3个目标
- 命中目标后**削弱攻击力10%，持续3秒**（`atkReducePercent=0.10, atkReduceDuration=3.0`）
- 与金刚掌对标: 金刚掌附带减速，破剑式附带减攻，均为0级状态效果

**奔雷式详细机制**:
- 灵剑从天而降，砸在面前圆形区域，造成 **150%** ATK 砸击伤害
- 落点插剑，持续电击周围敌人（**10% ATK/s × 5s，真实伤害无视防御**，电击范围2.0格）
- CD 5s（与破剑式相同，高频轮转）
- 连击时额外插剑（再增加一道DOT）
- 是太虚的持续DPS基底，高频释放 = 高频DOT叠加

**一剑开天详细机制**（筑基解锁被动）:
- 每释放 **8次** 技能（含连击触发），从天召唤巨剑
- **300%** ATK，面前圆形1.5格，最多5目标
- **双倍击杀回血**: 击杀目标时恢复的HP翻倍
- 不触发连击（被动巨剑不递归计数）
- 与龙象功对标: 龙象功是"18层普攻蓄力→AoE真伤"，一剑开天是"8次技能蓄力→300%爆发"
- 是太虚的核心爆发引擎

**御剑诀详细机制**（练气解锁被动）:
- 3柄灵剑环绕角色（视觉：绕身旋转的半透明剑影）
- 受到攻击时，1柄灵剑自动格挡，**减免该次伤害的30%**
- 每柄剑只能格挡1次，格挡后消失（3柄=最多格挡3次，总计减免3×30%=90%的伤害分散在3次受击中）
- **恢复机制**:
>   - 每 **20秒** 自动恢复1柄灵剑（上限3柄）
>   - 释放**任意同职业主动技能（含破剑式、奔雷式）**时额外恢复1柄灵剑（上限3柄）；一剑开天（`nth_cast_trigger`类型）不触发
- 与金钟罩对标: 金钟罩是"一次性大护盾(DEF×4)"，御剑诀是"分散在多次受击的减伤"
- **设计意图**: 太虚不再是"死了再说"的赌命风格，而是"持续但有限的防护"，更适合持续战斗

**DOT（电击）通用机制**（奔雷式产生的插剑）:
- 奔雷式命中后在目标区域插剑，小剑持续电击周围敌人（视觉：剑身放电，蓝白色闪电弧）
- 电击伤害 = ATK × 10% × (1+skillDmg%)/秒，**持续5秒**，**真实伤害无视防御**，**吃悟性技能伤害%**
- 电击范围 2.0格（略大于和尚金刚掌的1.5格命中范围）
- 每秒独立判暴击（走 ApplyCrit）
- **连击时额外插剑**（连击概率来源: 悟性+1%/50pt + 职业天赋+8%）
- 多道剑阵可以叠加（同一区域多柄小剑同时放电，DPS叠加），最多同时3道
- **注意**: 真实伤害无视防御，使DOT在高DEF敌人面前不被削弱

### 镇岳技能 🆕

| 技能 | 类型 | 效果 | CD |
|------|------|------|-----|
| 震山掌 | 主动 AoE | 150% ATK，近战范围，最多5目标 | 5s |
| 血气冲霄 | 主动 BUFF | ATK + maxHP×8%，持续10s | 20s |

**震山掌** — 血怒叠层加速器:
- 命中目标**必定叠加1层血怒**（无需概率判定）
- 5个目标 = 5层 = 直接引爆，镇岳在怪群中极其强势

## 四、罗汉 vs 太虚 DPS 计算（2026-04-14 更新）

> **v3 DPS 变更说明** (2026-04-14, v7.5更新):
> - 技能2: 一剑开天(主动300%CD10s) → **奔雷式**(主动150%砸击+DOT, CD5s)
> - 技能4: 剑阵降临(每次技能→DOT) → **一剑开天**(被动，每8次技能→300%巨剑)
> - 御剑诀不变（减伤被动，不计入DPS）

### 计算条件

| 参数 | 值 |
|------|-----|
| 玩家 ATK | 600 |
| 敌方 DEF | 400 |
| 暴击率 | 60% |
| 暴击伤害 | 450% (4.5×) |
| 重击值 (HeavyHit) | 400 |
| 基础 maxHP | 3000 |
| 根骨 | 100（罗汉每级+1，Lv100时） |
| 悟性 | 100（太虚每级+1，Lv100时） |
| 体魄 | 100（所有职业共有） |
| CalcDamage 公式 | ATK² / (ATK + DEF) |

> **关键机制**:
> - **重击/龙象功无视防御**: 伤害 = (ATK + HeavyHit) × (1+heavyDmgBonus)，直接 TakeDamage
> - **龙象功 = 1倍重击公式**，走ApplyCrit，AoE 4格最多8目标
> - **奔雷式DOT(v3)**: 砸击点创建DOT区域，10%/s×5s，rawDmg = ATK × 10% × (1+skillDmg%)/tick，**真实伤害无视防御**，每tick独立暴击
> - **一剑开天被动(v3)**: 每8次技能释放/连击触发300%巨剑，走CalcDamage+ApplyCrit，前方偏移
> - **各职业天赋在0pt即生效**: 罗汉重击+10%，太虚连击+8%
> - **罗汉无100pt被动**（金刚之力已移除）
> - **御剑诀不计入DPS**（减伤被动，不影响输出计算）

### 基础计算

```
CalcDamage(600, 400) = 360000/1000 = 360
期望暴击倍率 = 0.6×4.5 + 0.4×1.0 = 3.1
普攻 DPS（1.0s 间隔）= 360 × 3.1 = 1116/s
```

### 罗汉 DPS（100悟性 100根骨 100体魄）

天赋: 重击概率+10% | 技能: 金刚掌+伏魔刀+金钟罩+龙象功 | 无100pt被动

```
普攻 DPS: 1116/s

重击（无视防御！天赋10% + 100根骨→+4%概率 +4%伤害加成 = 14%概率）:
  重击dmg = (ATK + HeavyHit) × (1+heavyDmgBonus)
          = (600 + 400) × 1.04 = 1040
  重击期望 = 1040 × 3.1 = 3224
  概率 14%, 贡献 = 3224 × 0.14 = 451/s

龙象功（18层蓄力 → AoE真伤，单目标计算）:
  叠层速率: 普攻+1层/s + 重击14%×2层 = 1.28层/s → 14.1s叠满
  伤害公式 = 重击公式 = (600+400)×1.04 = 1040, ×3.1 = 3224
  单目标 DPS = 3224 / 14.1 = 229/s
  （多目标: ×8目标 = 1829/s，AoE优势巨大）

技能 DPS（100悟性 → skillDmgPercent=20%）:
  金刚掌: effATK=600×1.5×1.2=1080 → CalcDmg(1080,400)=788, ×3.1=2443, /5s = 489/s
  伏魔刀: effATK=600×1.0×1.2=720 → CalcDmg(720,400)=463, ×3.1=1435, /8s = 179/s（+全额吸血）
  金钟罩: 被动护盾，不计入DPS

总 DPS ≈ 1116 + 451 + 229 + 489 + 179 = 2464/s
（高生存: 吸血+护盾，龙象功多目标时DPS暴涨）
```

### 太虚 DPS（100悟性 100根骨 100体魄）— v3

天赋: 连击概率+8% | 技能: 破剑式+奔雷式(砸击+DOT)+御剑诀+一剑开天(被动N=8)

```
普攻 DPS: 1116/s

技能 DPS（100悟性 → skillDmgPercent=20%）:
  破剑式: effATK=600×1.5×1.2=1080 → CalcDmg(1080,400)=788, ×3.1=2443, /5s = 489/s
  奔雷式(砸击): effATK=600×1.5×1.2=1080 → CalcDmg(1080,400)=788, ×3.1=2443, /5s = 489/s
  御剑诀: 减伤被动，不计入DPS

奔雷式DOT（砸击点创建DOT区域，10%/s×5s，真实伤害无视防御）:
  每tick: rawDmg=600×0.10×1.2=72, ×3.1(暴击期望)=223/tick
  每道总伤: 223×5tick = 1116
  60s内: 奔雷式12次(CD5s) = 12道DOT
  DOT DPS: 12×1116/60 = 223/s

一剑开天被动（每8次技能/连击触发，300%巨剑，前方偏移）:
  60s内技能释放: 破剑式12次 + 奔雷式12次 = 24次
  连击事件: 24×10% = 2.4次（也计入N计数器）
  总N计数事件: 24+2.4 = 26.4
  触发次数: 26.4/8 = 3.3次
  effATK=600×3.0×1.2=2160 → CalcDmg(2160,400)=1823, ×3.1=5651
  被动DPS: 3.3×5651/60 = 311/s

连击（天赋8% + 100悟性+2% = 10%）:
  60s内: 24次技能 × 10% = 2.4次连击
```

## 七、设计原则备忘

1. **100点被动**: 单一效果，无"必定"，基于累积强化
2. **连击 vs 双倍伤害**: 连击（重新施放）体验优于双倍数字
3. **天衍**: DEF = 攻防一体属性，BUFF 机器定位
4. **太虚**: 高爆发但脆皮，击杀回血是唯一续航
5. **宠物狂暴**: 绑定技能 CD（聚灵阵），可预测、可操作
6. **重击值×2**: 只翻倍 HeavyHit 数值，不翻倍整个伤害
7. **镇岳**: HP 是攻防一体属性（血气冲霄转化ATK + 铁躯减伤 + 血怒引爆回血）
8. **血怒场景极化**: 刻意设计——普通刷怪时镇岳不突出，BOSS战时镇岳是MVP
