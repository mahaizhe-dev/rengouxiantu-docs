### 怪物进阶结构

- **小怪**：每寨独有种族（枯木精/岩蟾/苍狼/赤甲蝎/蛇骨妖/赤焰妖/蜃妖）
- **BOSS**：全部为妖王级，2~3阶段战斗，越往内战斗机制越复杂
- **精英巡逻**：三片荒漠各有沙暴妖兵（妖尉→妖将→妖帅），主线需击杀100个
- **枯木守卫**：黄天寨门卫（固定型BOSS，地刺远程攻击）
- **猪二哥**：隐藏神器BOSS，Lv.70帝首领，3阶段

### 主线任务（sand_zone）

1. **扫平外围** — 击败枯木妖王+岩蟾妖王 → T5紫装三选一
2. **深入腹地** — 击败苍狼妖王 + 沙漠精英×100 → T2技能书三选一
3. **肃清妖患** — 击败赤甲妖王+蛇骨妖王 → T6紫装三选一
4. **斩妖除魔** — 击败烈焰狮王 + 沙漠精英×100 → T2技能书三选一
5. **兵临城下** — 击败蜃妖王 + 沙漠精英×100 → T7紫装三选一
6. **解除封印** — 解锁第一寨大门

### 剧情脉络

沙漠中九寨妖王联盟盘踞，以黄天大圣·沙万里为首。玩家从最外围的废寨出发，逐寨攻破，每寨妖王有独特战斗风格。三片荒漠间有精英妖兵巡逻。最终在第一寨对决帝级妖王沙万里。隐藏BOSS猪二哥（神器级）潜伏于神器竞技场。

### 跨章挑战系统

- **血煞盟·沈墨**（T1~T8）：血印 debuff 机制，叠层持续掉血
- **浩气宗·陆青云**（T1~T8）：结界领域机制，持续伤害区域
- 挑战不属于特定章节，随等级解锁

### 设计特点

- **八卦阵法**：8个阵法区域按等级梯度递增（Lv.66→99），每阵对应一种元素属性，由太虚宗上古弟子坐镇
- **封印推进**：坎阵免费开放，后续七阵（艮→乾）逐步解锁，每击杀当前阵BOSS 10次解锁下一阵传送
- **四方龙种岛**：四角最高等级区域（Lv.100·大乘初期），4条帝级龙族BOSS，各有3阶段战斗和独特龙鳞材料
- **天罡四柱**：乾阵门卫（固定型BOSS，地刺远程），分别铭刻「心正」「意诚」「身端」「道明」
- **境界跨度**：元婴后期(Lv.66) → 大乘初期(Lv.100)，横跨化神、合体、大乘三个大境界
- **新增种族**：水傀、岩俑、雷魂、残影、焦骨、土俑、蛊毒、石柱等8种新种族修正

### 主线任务（bagua_zone）

8步顺序任务，逐阵推进：

1. **破坎阵** — 击败沈渊衣×10 → T7橙装三选一（解锁艮阵）
2. **破艮阵** — 击败岩不动×10 → T8紫装三选一（解锁震阵）
3. **破震阵** — 击败雷惊蛰×10 → 九转金丹×1 + 龙髓×1（解锁巽阵）
4. **破巽阵** — 击败风无痕×10 → T8橙装三选一（解锁离阵）
5. **破离阵** — 击败炎若晦×10 → T9紫装三选一（解锁坤阵）
6. **破坤阵** — 击败厚德生×10 → T9橙装三选一（解锁兑阵）
7. **破兑阵** — 击败泽归墟×10 → 高级攻/防/生技能书三选一（解锁乾阵）
8. **破乾阵** — 击败司空正阳×10 → 高级蕴灵技能二选一

### 剧情脉络

太虚宗万年前的上古修士被八卦阵法封印于海域群岛之中，阵力侵蚀下丧失自我，化为阵中守卫。八位阵主分别坐镇坎至乾八阵：沈渊衣（水）→ 岩不动（山）→ 雷惊蛰（雷）→ 风无痕（风）→ 炎若晦（火）→ 厚德生（地）→ 泽归墟（泽）→ 司空正阳（天，太虚宗大师兄）。四角龙种岛封印着四条上古龙族：封霜应龙、堕渊蛟龙、焚天蜃龙、蚀骨螭龙，均为大乘初期帝首领。

### 设计特征

| 特征 | 说明 |
|------|------|
| **一剑裂谷** | 天裂峡谷非自然形成，为上古大能一剑劈开。地形表现为笔直横贯东西的裂谷，边缘齐整，深不见底，仅三处桥梁可通行 |
| **仙府都城风格** | 青石长街、琉璃亭台、灵气瀑布、祥云缭绕。与前四章的村庄/堡垒/沙漠/海域完全不同的城市化风貌 |
| **枢纽而非战场** | 南部全安全区，无野怪刷新，定位为功能 NPC 聚集地和传送枢纽。战斗内容集中在北部裂谷外的三战场 |
| **青云城核心地位** | 青云城是最大区域（36×34），承载绝大多数功能 NPC（商店、仓库、锻造、传送等），其余五势力区域为特色功能预留 |
| **左右两翼对称** | 左翼：血煞盟+天机阁（纵向相连）；右翼：浩气宗+瑶池（纵向相连）。形成以青云城为核心的对称布局 |
| **三通道设计** | 仅上方三势力（血煞盟/封魔殿/浩气宗）有跨裂谷通道，暗示这三个势力承担着对抗北方威胁的前线职责 |
| **渐进开放** | 三战场分阶段解封：仙劫（合体期 Lv.100）→ 仙陨（渡劫期 Lv.140）→ 仙殒（大乘期 Lv.120），封印条件已实现 |
