## 二、下架操作清单

### 2.1 关闭掉落（一键生效）

```lua
-- EventConfig.lua
enabled = false  -- 掉落立即停止，服务端协议拒绝
```

**影响链路**：
- `RollEventDrops()` → 检查 `EventConfig.IsActive()` → 返回 false → 不掉落
- `HandleOpenFudai` → 入口 `IsActive()` → 直接 return
- `main.lua` NPC加载 → `eventBound` NPC 不加入 GameState.npcs

### 2.2 关闭黑市活动道具

从 `BlackMerchantConfig.ITEMS` 移除：
- `childday_rattle`
- `childday_pinwheel`

### 2.3 修改道具描述

`GameConfig.EVENT_ITEMS` 中两个六一道具的 `desc` 改为「活动已结束」提示。保留 `sellPrice` 不变，玩家仍可出售换金币。

### 2.4 兑换码处理

保留不改。兑换码已有 `maxUses` 限制，且道具本身仍然有出售价值（sellPrice=1000金），玩家兑换后可自行出售。

## 三、测试方案

### 3.1 下架验证测试

| # | 测试项 | 步骤 | 预期结果 |
|---|--------|------|---------|
| T1 | 掉落停止 | 击杀任意BOSS（各章节各category） | 不再掉落拨浪鼓/风车 |
