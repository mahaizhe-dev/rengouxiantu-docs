### 2.1 当前代码实装

代码基线：

- `scripts/config/TrialTowerConfig.lua`
- `scripts/systems/TrialTowerSystem.lua`
- `scripts/ui/TrialTowerUI.lua`
- `scripts/network/TrialHandler.lua`

当前结论：

- `TrialTowerConfig.FLOOR_MONSTERS` 实际只生成到 `180` 层。
- `TrialTowerConfig.MAX_DAILY_TIER = 18`，并且服务端 `TrialHandler.lua` 里也内联写死了 `18`。
- 实际试炼增幅是 `atkMult=1.5`、`hpMult=2.0`、`atkSpeedMult=1.3`、`speedMult=1.3`、`cdrMult=0.7`，比旧预设计稿更高。
- 进入试炼只校验境界，不额外校验等级。
- 怪物真实境界不是固定写死，而是由 `GameConfig.GetMaxRealmByLevel(floorLevel)` 反推。
- 每层首通即时发奖励；每日供奉按“里程碑追领优先，追满后再领当日最高档”执行。

当前代码公式：

```lua
realmIndex = math.ceil(floor / 10)
localFloor = ((floor - 1) % 10) + 1

monsterLevel = realm.reqLevel
    + math.floor((realm.maxLevel - realm.reqLevel) * (localFloor - 1) / 9)

firstClearLingYun = realmIndex * 2 + math.floor(localFloor / 3) + 1
firstClearGoldBar = math.max(1, math.floor((realmIndex * 500 + localFloor * 100) / 500 + 0.5))

dailyTier = math.min(math.floor(highestFloor / 10), MAX_DAILY_TIER)
dailyReward = floor10FirstClearOfTier * 2
```

## 3. 镇狱塔

### 3.1 当前代码实装

代码基线：

- `scripts/config/PrisonTowerConfig.lua`
- `scripts/systems/PrisonTowerSystem.lua`
- `scripts/ui/PrisonTowerUI.lua`
- `scripts/network/RankHandler.lua`
- `scripts/network/SaveWriteService.lua`

当前结论：

- 当前实际开放 `150` 层。
- 当前 BOSS 池固定 `30` 个，对应 `30 * 5 = 150` 层。
- 当前不是“所有已解锁 BOSS 混池轮换”，而是“新 BOSS + 上一段 BOSS”交接模型。
- 当前每层 BOSS 数量不是 `1/2/3/4/5`，而是 `1/2/3/4/4`。
- 当前塔柱不是每层都有，只在 `5 / 10 / 15 ...` 这类里程碑层生成。
- 里程碑层的塔柱不会替换 BOSS 位，而是在 `4` 只 BOSS 之外额外添加。
- 非里程碑层如果当前层只有 `1` 只 BOSS，这只 BOSS 会站在中心点；里程碑层中心点留给塔柱。
- 镇狱塔内 BOSS 品质不看原怪物类别，而是按 `PrisonTowerConfig.GetEffectiveCategory()` 结算，不低于 `皇级`。

当前代码公式：

```lua
segmentIndex = math.ceil(floor / 10)
localFloor = ((floor - 1) % 10) + 1

monsterLevel = realm.reqLevel
    + math.floor((realm.maxLevel - realm.reqLevel) * (localFloor - 1) / 9)

bossCount = math.min(((floor - 1) % 5) + 1, 4)   -- 1/2/3/4/4
unlockCount = math.min(30, math.ceil(floor / 5))

if firstFiveFloors then
    activeBossIds = { currentBoss repeated bossCount times }
else
    activeBossIds = { currentBoss repeated bossCount times, previousBoss filled to 4 }
end
```

当前固定参数：

- 入场要求：`REQUIRED_REALM_ORDER = 7`、`REQUIRED_LEVEL = 40`
- 镇狱增幅：`hp/atk/def/speed/atkSpeed = x2`、`cdr = 0.5`
- 塔柱 AOE：`damagePct = 0.6`，按参考 BOSS 攻击计算
- 里程碑：每 `5` 层一次
