## 1. 系统概述

### 1.1 变更摘要

| 维度 | 旧系统 | 新系统 |
|------|--------|--------|
| 品类 | 法宝 | **葫芦**（独立品类） |
| 技能槽位 | 占1个技能位 | 占1个技能位（不变） |
| 技能成长 | 技能随阶级成长（T1~T11） | **技能固定**，等级仅提升属性 |
| 升级逻辑 | — | 不变（消耗/等级上限保持原设计） |
| 核心扩展 | 无 | **美酒系统**（3个装备槽） |

### 1.2 核心规则

```
美酒总数：8 种（第一章×1  第二章×1  第三章×2  第四章×4）
装备槽位：3 个
所有美酒：一次性掉落，获得后永不再掉落
每种美酒：仅一种效果
```

### 1.3 美酒效果类型

| 效果类型 | 机制 | 示例 |
|---------|------|------|
| **被动型** | 装入槽位即永久生效 | 根骨+25 |
| **饮用型** | 释放葫芦技能时触发，**持续时间与葫芦技能一致** | 饮用期间攻击+10% |
| **冷却型** | 葫芦技能冷却期间持续生效 | CD期间防御+10% |
| **强化饮用型** | 强化葫芦技能本身参数 | 饮用持续时间+3秒 |

**关键规则：饮用型效果持续时间 = 葫芦技能持续时间**，不单独计时。装备漠商醇延长技能时间后，所有饮用型效果同步延长。

## 2. 葫芦本体

### 2.1 固定技能

技能不再随阶级变化。等级提升仅影响主属性（hpRegen）和副属性。

```
技能名：治愈
CD：    30秒
效果：  持续 10秒，每秒回复 4.0% 最大生命
总回复：40% 最大生命
```

### 2.2 升级系统

升级逻辑、消耗材料、等级上限：沿用原有设计，不做改动。

等级提升 → 主属性(hpRegen)增长 + 副属性增长 → 提供额外属性收益。

### 2.3 技能数值评估

| 对比项 | 数值 | 说明 |
|--------|------|------|
| 治愈总回复 | 40%HP | 有用但不满血，美酒有提升空间 |
| 金刚掌 | 150%ATK/5sCD | 输出型，高频 |
| 伏魔刀 | 100%ATK+吸血/8sCD | 攻防兼备 |
| 金钟罩 | DEF×4护盾/60sCD | 被动防御 |

治愈定位：**纯生存向，CD长效果中等，通过美酒补强到合适水平**。

## 3. 美酒系统

|------|------|---------|------|---------|

| 二 | 乌泉酿 | 乌万海 | 饮用瞬间回复15%生命 | 饮用型 |

| 三 | 漠商醇 | 商人出售(100W) | 饮用持续时间+3秒 | 强化饮用型 |
| 四 | 封霜龙酿 | 封霜应龙 | 根骨+25 | 被动型 |

### 3.2 内部掉率（不对玩家展示）

| 美酒 | 掉率 | 备注 |
|------|------|------|
| 虎骨酒 | 首杀100% | 系统更新后首次击杀虎王必掉 |
| 乌泉酿 | 10% | |
| 沙煞酒 | 0.5% | |
| 漠商醇 | 购买 | 100W金币，限购1瓶 |
| 封霜龙酿 | 0.5% | |
| 渊蛟龙酿 | 0.5% | |
| 焚天龙酿 | 0.5% | |
| 蚀骨龙酿 | 0.5% | |

所有美酒一次性掉落，获得后该怪物不再参与该美酒的掉率判定。无保底机制。

### 3.3 装卸规则

- 已获得的美酒可自由装入/卸下
- 同一种美酒不可重复装载
- 装卸无冷却、无消耗

### 3.4 饮用型效果时间规则

```
基础状态：
  葫芦技能持续 10秒
  虎骨酒效果持续 10秒（= 技能时间）
  乌泉酿瞬间生效（不受时间影响）

装备漠商醇后：
  葫芦技能持续 13秒（10+3）
  虎骨酒效果持续 13秒（同步延长）
  乌泉酿仍为瞬间生效
```

## 4. 美酒详表

### 4.1 第一章 —— 虎骨酒

> *"虎王筋骨浸烈酒，饮之虎气生，拳拳带风。"*

| 维度 | 内容 |
|------|------|
| **效果** | 饮用期间攻击+10%（饮用型，持续时间=技能时间） |
| **掉落目标** | 虎王 |
| **掉落规则** | 系统更新后首次击杀100%掉落，此后不再掉落 |

触发流程：
```
系统更新后首次击杀虎王
  → 掉落"虎骨"（赤金光效）
  → 拾取后葫芦夫人出现：
    "好一块虎骨！以此酿酒，饮时虎气加身，攻伐更猛。"
  → 美酒获得演出（赤金色）
  → 引导装入槽位
```

数值评估：
```
Lv.40 总ATK约400：+15% = +60 ATK，持续10秒
Lv.100 总ATK约1500：+15% = +225 ATK，持续10秒
覆盖率 10/30 = 33%，等效全程增伤 ≈ 5%
装备漠商醇后：覆盖率 13/30 = 43%，等效 ≈ 6.5%
```

### 4.2 第二章 —— 乌泉酿

> *"乌万海心头之血，化为幽泉。此酒入喉，伤口自愈，百骸回春。"*

| 维度 | 内容 |
|------|------|
| **效果** | 饮用瞬间回复15%最大生命值（饮用型·瞬时） |
| **掉落目标** | 乌万海 |
| **掉落规则** | 一次性掉落，获得后不再掉落 |

触发流程：
```
击杀乌万海 → 概率掉落"乌泉心血"
  → 葫芦夫人："乌万海心头精血……以此酿酒，饮时可续断骨、生死肉。"
  → 美酒获得演出（碧绿色）
```

数值评估：
```
葫芦技能固定回复 40%HP
加乌泉酿：40% + 15% = 55%HP
提升幅度合理，不至于满血但显著改善生存
```

### 4.3 第三章 —— 沙煞酒

> *"沙暴之中有妖帅炼煞为酒，此酒入腹，刀枪难侵。"*

| 维度 | 内容 |
|------|------|
| **效果** | 饮用期间免疫击晕和击退（饮用型） |
| **掉落目标** | 沙暴妖帅 |
| **掉落规则** | 一次性掉落，获得后不再掉落 |

触发流程：
```
击杀沙暴妖帅 → 概率掉落"煞沙凝珀"
  → 葫芦夫人："妖帅百年修为凝成此酒。饮后余韵不散，刀枪难近。"
  → 美酒获得演出（暗金+沙尘色）
```

数值评估：
```
CD 30秒，技能持续10秒，CD期间 = 20秒
覆盖率 20/30 = 67%
等效全程防御提升 ≈ 6.7%
装备漠商醇后：CD期间 = 17秒，覆盖率 17/30 = 57%，等效 ≈ 5.7%
（漠商醇延长饮用时间会略微压缩CD期间，影响不大）
```

### 4.4 第三章 —— 漠商醇

> *"西域商人走遍万里黄沙，唯此一壶，千金不换。"*

| 维度 | 内容 |
|------|------|
| **效果** | 饮用持续时间+3秒（强化饮用型） |
| **获取** | 第三章NPC商人，100W金币，限购1瓶 |

触发流程：
```
商人对话：
  "这壶酒，100万金。贵？它值。"
  [购买 - 1,000,000金币]
  [改日再来]
  → 购买后葫芦夫人："好酒！酒劲悠长，饮一口，回味更久。"
  → 美酒获得演出（琥珀色）
已购买后商品显示"已售罄"。
```

数值评估：
```
基础技能持续 10秒 → 加漠商醇后 13秒
治愈回复：10×4% = 40%HP → 13×4% = 52%HP（+12%HP）
虎骨酒覆盖率：33% → 43%（+10%覆盖）
沙煞酒：饮用期间免疫控制，饮用时间越长免疫窗口越大

核心价值：放大葫芦技能本身 + 联动强化其他饮用型美酒
```

### 4.5 第四章 —— 四龙酿

四种酒分别掉落自第四章八卦海四兽岛的四方龙族皇级BOSS（Lv.100），全部为被动型，对应四种仙缘属性。

## 8. 搭配策略

### 8.1 效果类型分布

```
被动型 ×4（四龙酿）—— 装上就有，无需操作
饮用型 ×3（虎骨酒、乌泉酿、沙煞酒）—— 释放技能时触发（沙煞酒为免疫击晕和击退）
强化饮用型 ×1（漠商醇）—— 放大技能和饮用型效果
```

8种酒选3个，C(8,3) = 56种组合。

### 8.2 典型流派

| 流派 | 搭配 | 效果汇总 |
|------|------|---------|
| **爆发饮酒** | 虎骨酒 + 乌泉酿 + 漠商醇 | 饮用13秒：攻击+10%、瞬回15%HP、治愈52%HP |
| **攻防交替** | 虎骨酒 + 沙煞酒 + 漠商醇 | 饮用13秒：攻击+10%、免疫击晕击退，治愈52%HP |
| **续航生存** | 乌泉酿 + 沙煞酒 + 焚天龙酿 | 瞬回15%HP，饮用时免疫击晕击退，体魄+25常驻 |
| **武僧专精** | 封霜龙酿 + 沙煞酒 + 焚天龙酿 | 根骨+25、体魄+25常驻，饮用时免疫击晕击退 |
| **剑修专精** | 渊蛟龙酿 + 虎骨酒 + 漠商醇 | 悟性+25常驻(技伤+5%)，饮用13秒攻击+10% |
| **阵法师专精** | 蚀骨龙酿 + 乌泉酿 + 沙煞酒 | 福缘+25常驻，瞬回15%HP，饮用时免疫击晕击退 |
| **体修专精** | 焚天龙酿 + 乌泉酿 + 漠商醇 | 体魄+25常驻，瞬回15%HP，治愈52%HP |

### 8.3 饮用型搭配博弈

```
纯爆发（虎骨+乌泉+漠商）：
  → 饮酒13秒内极强（攻击+10%、瞬回+持续回复），CD期间无加成
  → 适合：Boss战关键时刻爆发

攻防兼备（虎骨/乌泉+沙煞+漠商/被动）：
  → 饮酒期间既有攻击/回复增益，又免疫控制
  → 适合：高难怪物频繁使用晕眩/击退的场景

被动+免控（沙煞+两个被动）：
  → 常驻属性加成，饮酒时额外免疫控制
  → 适合：持久战、需要稳定不被打断的续航打法
```

### 9.3 掉落判定伪代码

```lua
function OnMonsterKilled(monster_id)
    -- 遍历所有美酒配置
    for _, wine in ipairs(WineDefinitions) do
        if wine.obtain.source_type ~= "kill" then goto continue end
        if wine.obtain.source_id ~= monster_id then goto continue end

        -- 已获得则跳过（一次性掉落）
        if player.wine_data.obtained[wine.wine_id] then goto continue end

        -- 首杀类型特判
        if wine.obtain.first_kill then
            if player.gourd_first_kill_flags[monster_id] then goto continue end
            player.gourd_first_kill_flags[monster_id] = true
            -- 首杀100%掉落，不走概率
            DropWine(wine)
            goto continue
        end

        -- 概率判定
        if math.random() < wine.obtain.drop_rate then
            DropWine(wine)
        end

        ::continue::
    end
end

function DropWine(wine)
    player.wine_data.obtained[wine.wine_id] = true
    -- 触发葫芦夫人出现 + 美酒获得演出
    TriggerWineAcquisition(wine)
end
```
