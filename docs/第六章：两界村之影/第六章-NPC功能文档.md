## 二、设计原则

1. **不污染初章**：第六章不得复用第一章 NPC、装饰和刷怪点；如果功能相同，也必须使用第六章独立 ID。
2. **ID 前缀统一**：NPC 与交互对象使用 `ch6_` 前缀，仓库类对象使用 `warehouse_chest_ch6`。
3. **显示口径使用新仙阶**：NPC 文案、任务文案、功能说明使用逐怪物仙阶口径；本章 `人仙1阶` 为 140 级压轴怪口径，不再使用旧三段境界描述。
4. **系统入口先行**：先保证传送、主线、炼丹炉、仓库、基础商店等章节基础功能，再扩展特殊玩法。
5. **设计与落地分层**：本文可以先记录“候选 NPC / 待定功能”，只有进入代码时才写入 `ZoneData_ch6` 或后续拆分的第六章 zone 文件。

## 四、主线 NPC 功能

当前第六章主线建议按 7 步建立，主线 NPC 负责发放、进度展示、奖励领取和章节完成标记。

| 步骤 | 任务名 | 功能目标 | 关键目标 |
|------|--------|----------|----------|
| 1 | 影入口试锋 | 建立天界封锁线认知 | 击败巡游天神·凌风 |
| 2 | 夜荒寻踪 | 打开村北旧场线索 | 清理影游使，击败夜游神·烛幽 |
| 3 | 断岳开山 | 连接后山与东西大营 | 击败两界山神·断岳 |
| 4 | 破西营 | 突破西侧天界军阵 | 击败破军、镇垣、哼元帅 |
| 5 | 破东营 | 突破东侧天界军阵 | 击败青锋、雷策、哈元帅 |
| 6 | 呱声旧梦 | 回收两界村荒诞旧线 | 击败蛤蟆仙人、呱大人 |
| 7 | 村心蚀玄 | 完成第六章终局 | 击败魔君·蚀玄，解除村心封印 |

## 五、落地注意

| 模块 | 注意点 |
|------|--------|
| `ZoneData_ch6.lua` | 当前显式清空 NPC，不要误以为第一章 NPC 会自动出现 |
| `ChapterConfig.lua` | `minRealm = "dujie_1"` 是兼容字段，文案层应显示 `谪仙1阶` |
| `GameConfig.CONSUMABLES` | `一转仙劫丹` 已有物品定义，描述含第六章炼丹炉来源 |
| `AscensionConfig.lua` | 已有第六章炼丹炉消耗配置 |
| UI | 炼丹炉如果复用旧 `alchemy`，需要根据章节或配方 ID 切换产物 |
| 存档 | 主线进度、新炼丹入口、终局封印状态需要独立 key，不复用第一章主线 key |
