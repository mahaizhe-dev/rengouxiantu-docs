# 第五章 · 四仙剑战斗设计

> 用途：给后续 AI 直接评估与执行的独立战斗设计稿。  
> 原则：只写战斗结构、技能清单、阶段逻辑，不写数值平衡。

---

## 一、总规则

### 1.1 四仙剑共通规则

| 项目 | 设定 |
|------|------|
| Boss形态 | 四把仙剑均为 `固定Boss`，锁在各自封剑台中央。 |
| 位移规则 | `不能移动、不能追击、不能离台`。 |
| 攻击方式 | 只通过 `剑气、剑阵、剑痕、场地变化、临时机制体` 攻击玩家。 |
| 召唤限制 | 不刷新常规小怪；允许出现 `镜面、火柱、血印、剑影` 这类临时机制体。 |
| 战斗目标 | 玩家体验应是 `读招、找站位、拆机制、处理场地压力`，不是绕圈拉Boss。 |
| 阶段结构 | 四把仙剑统一采用 `两阶段` 结构，`生命降至70%` 时进入转阶段。 |
| 二阶段原则 | 二阶段不推翻一阶段玩法，只在原有剑意上加压。 |

### 1.2 统一技能结构

每把仙剑固定拥有以下结构：

- `1个常驻被动`
- `4个常规循环技能`
- `1个转阶段技能`
- `1个二阶段新增核心技能`

## 二、诛仙剑

### 2.1 战斗定位

| 项目 | 设定 |
|------|------|
| 剑意 | 纯正面压制、斩线处决 |
| 体验关键词 | `快节奏`、`读正反斩`、`短窗口爆发` |
| 封剑台 | 狭长主台，左右各有一条短侧带可暂避，前后空间开阔 |

### 2.2 技能清单

| 技能名 | 类型 | 描述 | 用处 |
|------|------|------|------|
| 剑心锁域 | 常驻被动 | 诛仙剑始终面向当前仇恨目标，每次施法前先完成转向。Boss本体不移动，只允许原地转向。 | 保证所有斩线技能都围绕“正面”展开。 |
| 一字诛线 | 常规技能 | 诛仙剑向正前方打出一条超长直线剑气，地面先出现细长预警线，短延迟后斩落。 | 基础正面读招。 |
| 横天断命 | 常规技能 | 诛仙剑在自身左右展开横向斩击，覆盖主台中段，迫使玩家离开中轴。 | 打掉只会站正面的习惯。 |
| 十字绝路 | 常规技能 | 先落一次纵斩，再落一次横斩，两道斩线在Boss脚下交叉，第二段比第一段略慢。 | 形成连续两次换位。 |
| 追命剑标 | 常规技能 | 点名一名玩家，脚下生成剑印；短延迟后，剑印所在位置连续落下三次窄线斩击，最后一击范围更宽。 | 做点名处决，逼玩家主动带位。 |
| 剑宫鸣杀 | 转阶段技能 | 生命降至70%时触发。诛仙剑悬空震鸣，主台中央落下三道连续纵斩，阶段结束后左右短侧带缩窄。 | 明确转阶段，并压缩安全站位。 |
| 诛心归一 | 二阶段核心技能 | 诛仙剑在短蓄力后，仅保留一条极窄安全带，其余区域被整面斩过；安全带会在 `左 / 中 / 右` 三种位置中随机一种出现。 | 二阶段终极招，考验读图与瞬时走位。 |

### 2.3 技能循环逻辑

- 一阶段循环：`一字诛线 -> 横天断命 -> 十字绝路 -> 追命剑标`
- 二阶段循环：`一字诛线 / 横天断命 / 十字绝路 / 追命剑标` 中插入 `诛心归一`
- 设计限制：
  - 不加入减速、冻结、持续掉血
  - 不加入过多随机点名
  - 核心始终是 `斩线判断`

## 三、陷仙剑

### 3.1 战斗定位

| 项目 | 设定 |
|------|------|
| 剑意 | 冰镜折返、迟滞误导 |
| 体验关键词 | `中慢节奏`、`镜像读图`、`减速压迫` |
| 封剑台 | 圆形寒镜台，外圈为冰面，台面存在数块可反射剑光的镜区 |

### 3.2 技能清单

| 技能名 | 类型 | 描述 | 用处 |
|------|------|------|------|
| 寒镜凝域 | 常驻被动 | 战斗场地外圈始终覆盖冰面。站在冰面上的玩家移动会产生惯性滑步。 | 让走位判断始终带有迟滞感。 |
| 霜环迟行 | 常规技能 | 以Boss为圆心放出一圈扩散寒环，被命中者获得显著减速。 | 基础控场技能。 |
| 镜面反斩 | 常规技能 | 陷仙剑先向一个方向射出剑光，剑光命中镜区后会从镜区再反射一次。第一段和反射段有清晰先后。 | 核心识别技能。 |
| 冰牢点名 | 常规技能 | 点名一名玩家，原地生成冰牢。冰牢在短延迟后闭合并爆裂，闭合前玩家必须离开标记位置。 | 用点名制造离位压力。 |
| 千镜回锋 | 常规技能 | 场地边缘依次亮起数面寒镜，每面寒镜依次朝主台中心打出直线剑光。 | 形成顺序型躲避题。 |
| 寒台翻镜 | 转阶段技能 | 生命降至70%时触发。镜台翻转，新增两块镜区，外圈冰面扩大。 | 明确进入更复杂的镜面阶段。 |
| 万镜同坠 | 二阶段核心技能 | 全场所有镜区同时预警，随后按固定顺序依次反射剑光；玩家需要连续读三轮反射方向。 | 二阶段主要灭团技。 |

### 3.3 技能循环逻辑

- 一阶段循环：`霜环迟行 -> 镜面反斩 -> 冰牢点名 -> 千镜回锋`
- 二阶段循环：在原循环中定期插入 `万镜同坠`
- 设计限制：
  - 不做高爆发秒杀型Boss
  - 压力来自 `迟滞 + 误导 + 连续读图`
  - 镜面反射必须始终可读，不做纯随机乱射

## 四、戮仙剑

### 4.1 战斗定位

| 项目 | 设定 |
|------|------|
| 剑意 | 地火焚场、熔线切台 |
| 体验关键词 | `中快节奏`、`封场`、`空间切割` |
| 封剑台 | 破裂熔台，台面遍布裂口与断桥，部分区域可被地火临时吞没 |

### 4.2 技能清单

| 技能名 | 类型 | 描述 | 用处 |
|------|------|------|------|
| 炉火灼界 | 常驻被动 | 场地上存在若干裂口，部分技能会点燃裂口，使其在一段时间内变为不可站立区。 | 让场地持续恶化。 |
| 地火喷口 | 常规技能 | 随机选取数个裂口喷发地火，喷口前地面有圆形预警。 | 基础封点技能。 |
| 熔环外推 | 常规技能 | 以Boss为中心放出一圈向外扩散的火环，火环经过处留下短时灼地。 | 逼迫玩家提早换位。 |
| 落焰剑雨 | 常规技能 | 在玩家所在区域连续落下多轮火焰剑雨，单次范围不大，但会连续覆盖。 | 防止原地站桩。 |
| 断桥熔线 | 常规技能 | 戮仙剑朝数条固定方向放出熔线，熔线将台面切成多个区域，被切中的区域短时间持续灼烧。 | 核心切场技能。 |
| 炉心崩裂 | 转阶段技能 | 生命降至70%时触发。主台额外裂开一圈新裂口，场地永久减少一部分可站立面积。 | 明确转阶段，并强化封场主题。 |
| 八荒焚路 | 二阶段核心技能 | 戮仙剑一次性向八个方向放出熔线，只保留少量夹缝安全区；熔线结束后留下持续灼地。 | 二阶段最大站位考题。 |

### 4.3 技能循环逻辑

- 一阶段循环：`地火喷口 -> 熔环外推 -> 落焰剑雨 -> 断桥熔线`
- 二阶段循环：在原循环中定期插入 `八荒焚路`
- 设计限制：
  - 不依赖点名秒人
  - 不依赖减速和冻结
  - 核心始终是 `场地封锁`

## 五、绝仙剑

### 5.1 战斗定位

| 项目 | 设定 |
|------|------|
| 剑意 | 吸能蚀血、禁疗压血 |
| 体验关键词 | `慢压迫`、`血线管理`、`持续消耗` |
| 封剑台 | 血池暗台，边缘为蚀血区，中央区域安全但承压最高 |

### 5.2 技能清单

| 技能名 | 类型 | 描述 | 用处 |
|------|------|------|------|
| 血池侵神 | 常驻被动 | 场地边缘长期存在蚀血区，玩家停留其上会持续损血。 | 强制玩家尽量留在主战区。 |
| 吸血锁链 | 常规技能 | 绝仙剑向目标连出血链，短时间后开始吸血；玩家离开锁链范围可提前挣断。 | 做血线与拉位压力。 |
| 蚀心剑印 | 常规技能 | 在玩家脚下留下血印，血印延迟爆裂并附加一层蚀血印记；印记会强化后续技能伤害。 | 构成印记累积机制。 |
| 禁疗血雾 | 常规技能 | 在场地局部生成血雾区，站在其中时治疗效果显著下降。 | 迫使玩家更主动地规划站位。 |
| 压血断生 | 常规技能 | 绝仙剑对主台中心落下一道高压血斩，命中后额外按当前生命比例结算伤害。 | 专门打高血线站桩。 |
| 血池倒灌 | 转阶段技能 | 生命降至70%时触发。中央血池上涌，场地上新增若干缓慢移动的蚀血波纹。 | 明确转阶段，并把消耗压力持续化。 |
| 绝命蚀界 | 二阶段核心技能 | 全场进入短时血界，边缘与中央同时高压，场上只会出现少量净血安全点；玩家必须主动换位抢点。 | 二阶段的终局压迫技。 |

### 5.3 技能循环逻辑

- 一阶段循环：`吸血锁链 -> 蚀心剑印 -> 禁疗血雾 -> 压血断生`
- 二阶段循环：在原循环中定期插入 `绝命蚀界`
- 设计限制：
  - 不做铺天盖地的弹幕Boss
  - 不做高频瞬杀
  - 核心始终是 `控血、避雾、拆印记`

## 六、实现口径

### 6.1 给后续AI的固定约束

- 四把仙剑都是 `原地Boss`
- 不做位移技能
- 不做追击技能
- 不刷常规小怪
- 每把剑固定 `1被动 + 4常规 + 1转阶段 + 1二阶段核心`
- 四把剑的差异必须来自：
  - `台子形状`
  - `危险区结构`
  - `技能节奏`
  - `剑意主题`

### 6.2 当前已经定死的体验分工

| 神剑 | 体验核心 |
|------|------|
| 诛仙剑 | 正面斩线、快节奏爆发 |
| 陷仙剑 | 冰镜迟滞、连续读图 |
| 戮仙剑 | 地火封场、空间切割 |
| 绝仙剑 | 蚀血禁疗、血线压迫 |

### 6.3 这一份文档还不处理什么

- 具体伤害数值
- 技能冷却秒数
- 动画帧数
- 特效资源编号
- 仇恨算法细节
