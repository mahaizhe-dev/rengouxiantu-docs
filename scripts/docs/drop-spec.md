## 二、掉落原则

### 2.1 食物掉落规则

1. **BOSS 不掉肉骨头** -- 最低级食物不配出现在 BOSS 掉落中
2. **普通怪进化路线**：
   - 1章普通怪：最低食物 = 肉骨头，当前规范化为单一食物 roll
   - 2章普通怪：最低食物 = 肉骨头，当前规范化为单一食物 roll
3. **2章 BOSS 不掉灵肉** -- 常规 BOSS 从灵兽肉起步，乌万海使用仙骨
4. **精英怪**：比同级普通怪多一档食物，掉率更高

### 2.2 装备掉落规则

1. **普通怪**：10-15% 掉落率，品质范围 white~green（1章）/ white~purple（2章）
2. **精英怪**：100% 掉落 1 件，品质范围 white~blue（1章）/ white~purple（2章）/ white~orange（高级）
3. **BOSS**：100% 掉落 1 件高品质 + 概率掉落专属装备
4. **王级 BOSS**：100% 掉落 + 多件专属概率 + 套装

### 2.3 其他掉落规则

1. **灵韵**：BOSS 20-100% 单一 roll 掉落，精英不掉，普通怪不掉
2. **灵兽丹**：BOSS 概率掉落，对应当前章节突破需求
3. **练气丹/功法书**：特定 BOSS 掉落
